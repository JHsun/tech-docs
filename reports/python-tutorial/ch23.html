CHAPTER 23 · PART 7

網路程式設計

Python 是網路應用的王者之一 —— 從簡單的 API 呼叫到 WebSocket 即時通訊、從爬蟲到分散式系統。本章涵蓋 HTTP 協議基礎、requests、httpx（同步+異步）、aiohttp、WebSocket、以及底層 socket 的使用。

Python 3.14 HTTP · WebSocket · Socket 預估閱讀 25 分鐘

§0章節導論

網路程式設計分幾個層次：

  • 高層 HTTP 客戶端：requests（舊經典）、httpx（現代）、aiohttp（全 async）
  • WebSocket 即時通訊：websockets 套件
  • 標準庫：urllib（預設但笨）、http.client、http.server
  • 底層 socket：自己實作 TCP/UDP 協議時才用到

本章聚焦實戰用得到的 httpx、aiohttp、websockets，底層 socket 只做概念介紹。

§23.1HTTP 協議基礎

HTTP 方法

方法用途冪等請求體
GET讀取資源✓無
POST建立資源✗有
PUT完整更新✓有
PATCH部分更新✗有
DELETE刪除資源✓無/有
HEAD只取 headers✓無
OPTIONS查詢支援方法✓無

狀態碼

範圍意義常見
1xx資訊100 Continue, 101 Switching Protocols
2xx成功200 OK, 201 Created, 204 No Content
3xx重新導向301 Moved, 302 Found, 304 Not Modified
4xx客戶端錯誤400 Bad Request, 401 Unauthorized, 403 Forbidden, 404 Not Found, 429 Rate Limit
5xx伺服器錯誤500 Internal Error, 502 Bad Gateway, 503 Unavailable

§23.2requests —— 老字號同步 HTTP

雖然 httpx 功能更強，但 requests 仍是社群最廣泛的入門工具。

python 
# uv add requests
import requests

# GET
r = requests.get("https://api.github.com/users/python")
print(r.status_code)        # 200
print(r.json())             # dict
# print(r.text)              # str
# print(r.content)           # bytes

# Query string
r = requests.get("https://api.github.com/search/users",
                 params={"q": "python", "per_page": 10})

# Headers
r = requests.get(url, headers={
    "User-Agent": "MyApp/1.0",
    "Authorization": "Bearer xyz",
})

# POST JSON
r = requests.post("https://api.example.com/items",
                  json={"name": "Alice", "age": 30})

# POST form
r = requests.post(url, data={"user": "alice"})

# 檔案上傳
with open("photo.jpg", "rb") as f:
    r = requests.post(url, files={"photo": f})

# 超時（永遠設！）
r = requests.get(url, timeout=10)
r = requests.get(url, timeout=(3, 10))   # (連線, 讀取)

# 檢查錯誤
r.raise_for_status()       # 4xx/5xx 拋 HTTPError

# Session（重用連線，cookie 自動保存）
with requests.Session() as s:
    s.headers.update({"Authorization": "Bearer xyz"})
    r1 = s.get("...")
    r2 = s.get("...")     # 使用同一個 TCP 連線，快很多
⚠️ 永遠設 timeout！

requests.get(url) 若沒指定 timeout，預設會無限等待！伺服器掛了會讓你的程式卡死。一律加 timeout=10 或 (3, 10)。

§23.3httpx —— 現代 HTTP（推薦）

httpx 是 requests 作者後來設計的現代替代品：API 與 requests 類似、但同時支援同步與異步、支援 HTTP/2、有更好的型別提示。FastAPI 的官方推薦客戶端。

python 
# uv add httpx
import httpx

# 同步 API（幾乎與 requests 相同）
r = httpx.get("https://api.github.com/users/python")
print(r.json())

# Client（相當於 requests.Session）
with httpx.Client(timeout=10, headers={"User-Agent": "MyApp"}) as client:
    r1 = client.get(url1)
    r2 = client.post(url2, json={"x": 1})

# HTTP/2 支援
client = httpx.Client(http2=True)   # 需額外安裝 h2

異步用法（重點）

python 
import asyncio
import httpx

async def fetch(url, client):
    r = await client.get(url)
    return r.json()

async def main():
    urls = ["https://httpbin.org/get"] * 10

    # AsyncClient 自動管理連線池
    async with httpx.AsyncClient(timeout=10) as client:
        async with asyncio.TaskGroup() as tg:
            tasks = [tg.create_task(fetch(u, client)) for u in urls]

    results = [t.result() for t in tasks]
    print(len(results))

asyncio.run(main())

# httpx 也支援 streaming 大檔下載
with httpx.Client() as client:
    with client.stream("GET", "https://example.com/big.zip") as r:
        with open("big.zip", "wb") as f:
            for chunk in r.iter_bytes(8192):
                f.write(chunk)

§23.4aiohttp —— 老牌異步 HTTP

aiohttp 既是 HTTP 客戶端，也是 server 框架。歷史比 httpx 久，爬蟲框架（scrapy 等）整合較成熟。

python 
# uv add aiohttp
import asyncio
import aiohttp

async def fetch(session, url):
    async with session.get(url) as r:
        return await r.text()

async def main():
    timeout = aiohttp.ClientTimeout(total=10)
    async with aiohttp.ClientSession(timeout=timeout) as session:
        html = await fetch(session, "https://python.org")
        print(len(html))

asyncio.run(main())

# JSON API
async with session.get(url) as r:
    data = await r.json()

# POST
async with session.post(url, json={"x": 1}) as r: ...
💡 httpx vs aiohttp 怎麼選
  • httpx：同步+異步一致 API、API 最現代、型別好、新專案首選
  • aiohttp：純異步、生態成熟、爬蟲框架多整合、效能稍高
  • 如果專案有同步部分共用 HTTP client，用 httpx

§23.5認證機制

python 
import httpx

# 1. Basic Auth
r = httpx.get(url, auth=("user", "password"))

# 或顯式
from httpx import BasicAuth
r = httpx.get(url, auth=BasicAuth("user", "pass"))

# 2. Bearer Token（OAuth2 / JWT）
headers = {"Authorization": f"Bearer {token}"}
r = httpx.get(url, headers=headers)

# 3. API Key in header
headers = {"X-API-Key": api_key}

# 4. API Key in query string
params = {"api_key": api_key}

# 5. OAuth2（通常用套件如 authlib、oauthlib）

# 6. 自訂 Auth class（httpx）
class CustomAuth(httpx.Auth):
    def __init__(self, token):
        self.token = token
    def auth_flow(self, request):
        request.headers["X-Custom-Auth"] = self.token
        yield request

r = httpx.get(url, auth=CustomAuth("xyz"))
⚠️ 密鑰管理
  • 永遠不要把 API key 硬編碼在程式碼！
  • 放環境變數：os.environ["API_KEY"]
  • 或用 python-dotenv 讀 .env 檔（加入 .gitignore）
  • 生產環境用 secret manager（AWS Secrets Manager、Vault）

§23.6WebSocket —— 即時雙向通訊

HTTP 是「請求-回應」模式，WebSocket 則是建立長連線後雙方都能主動發訊息 —— 聊天室、即時看板、遊戲、協同編輯的首選協議。

python · 客戶端 
# uv add websockets
import asyncio
import websockets

async def client():
    url = "wss://echo.websocket.events"
    async with websockets.connect(url) as ws:
        await ws.send("Hello")
        response = await ws.recv()
        print(response)

        # 持續收訊息
        async for message in ws:
            print(f"收到: {message}")

asyncio.run(client())
python · 伺服器 
import asyncio
import websockets

async def echo(ws):
    async for message in ws:
        await ws.send(f"echo: {message}")

async def main():
    async with websockets.serve(echo, "localhost", 8765):
        await asyncio.Future()      # 永遠執行

asyncio.run(main())

多用戶聊天室

python 
import asyncio, websockets
import json

clients = set()    # 所有連線的客戶端

async def broadcast(message):
    if clients:
        await asyncio.gather(
            *[c.send(message) for c in clients],
            return_exceptions=True
        )

async def handler(ws):
    clients.add(ws)
    try:
        await broadcast("使用者加入")
        async for msg in ws:
            await broadcast(msg)
    finally:
        clients.discard(ws)
        await broadcast("使用者離開")

async def main():
    async with websockets.serve(handler, "localhost", 8765):
        await asyncio.Future()

asyncio.run(main())

§23.7socket 底層（概念認識）

高階 HTTP 套件底層就是 socket。多數情況你不需要直接用，但了解概念有助於除錯。

python · TCP server 
import socket

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    s.bind(("localhost", 9999))
    s.listen()
    print("等待連線...")
    conn, addr = s.accept()
    with conn:
        print(f"來自 {addr}")
        while True:
            data = conn.recv(1024)
            if not data: break
            conn.sendall(data)      # echo
python · TCP client 
with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    s.connect(("localhost", 9999))
    s.sendall(b"Hello")
    data = s.recv(1024)
    print(data)

用 asyncio 的高階 socket API 更好管理：

python 
import asyncio

async def handle(reader, writer):
    data = await reader.read(1024)
    writer.write(b"echo: " + data)
    await writer.drain()
    writer.close()

async def main():
    server = await asyncio.start_server(handle, "localhost", 9999)
    async with server:
        await server.serve_forever()

asyncio.run(main())

§23.8簡易 HTTP 伺服器

bash 
# 標準庫一行 HTTP server（測試、分享檔案好用）
python -m http.server 8000

# 指定目錄
python -m http.server 8000 --directory /tmp

# 只綁本機
python -m http.server 8000 --bind 127.0.0.1
python 
# 自訂簡易 HTTP 處理器
from http.server import BaseHTTPRequestHandler, HTTPServer
import json

class MyHandler(BaseHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.send_header("Content-Type", "application/json")
        self.end_headers()
        self.wfile.write(json.dumps({"ok": True}).encode())

server = HTTPServer(("localhost", 8000), MyHandler)
server.serve_forever()

§23.9實戰技巧

重試 + 指數退避

python 
import httpx
import time

def get_with_retry(url, max_retries=3):
    for attempt in range(max_retries):
        try:
            r = httpx.get(url, timeout=10)
            r.raise_for_status()
            return r.json()
        except (httpx.TimeoutException, httpx.HTTPStatusError) as e:
            if attempt == max_retries - 1:
                raise
            wait = 2 ** attempt       # 1s, 2s, 4s, ...
            time.sleep(wait)

# 或用專業套件 tenacity
from tenacity import retry, stop_after_attempt, wait_exponential

@retry(stop=stop_after_attempt(3),
       wait=wait_exponential(multiplier=1, min=1, max=10))
def fetch(url):
    return httpx.get(url, timeout=10).json()

分頁遍歷

python 
def paginate(url, **params):
    """GitHub 風格的 Link header 分頁"""
    page = 1
    while url:
        params["page"] = page
        r = httpx.get(url, params=params, timeout=10)
        yield from r.json()

        # 解析 Link header 的 rel="next"
        if "next" in r.links:
            url = r.links["next"]["url"]
        else:
            break
        page += 1

for item in paginate("https://api.github.com/users/python/repos"):
    print(item["name"])
