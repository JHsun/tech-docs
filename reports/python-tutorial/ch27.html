CHAPTER 27 · PART 8

資料視覺化

Python 的視覺化生態分三層：matplotlib（所有工具的基石）、seaborn（統計分析專用）、plotly（互動式儀表板）。本章讓你知道何時用哪個、核心 API 如何寫，以及如何做出讓人看得懂、願意看的圖。

Python 3.14 matplotlib · seaborn · plotly 預估閱讀 22 分鐘

§0章節導論

工具定位適用
matplotlib底層繪圖引擎精細客製化、論文圖、靜態報表
seabornmatplotlib 之上的統計包裝統計關係視覺化、預設美觀
plotly互動式圖表儀表板、網頁嵌入、動態資料
pandas/polars .plot()快速繪圖探索性分析（EDA）
其他bokeh、altair、plotnine各有所長，非主流

§27.1matplotlib 基礎

bash 
uv add matplotlib

兩種 API 風格

python 
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

# ❌ pyplot state-machine 風格（方便但不推薦用於正式專案）
plt.plot(x, y)
plt.title("Sine")
plt.xlabel("x")
plt.show()

# ✅ Object-Oriented 風格（推薦）
fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(x, y, label="sin(x)", linewidth=2)
ax.set_title("Sine function")
ax.set_xlabel("x")
ax.set_ylabel("sin(x)")
ax.legend()
ax.grid(True, alpha=0.3)
plt.show()

# 儲存圖片
fig.savefig("sine.png", dpi=150, bbox_inches="tight")
fig.savefig("sine.pdf")           # 向量格式（論文適用）
fig.savefig("sine.svg")           # 可編輯向量
💡 fig 與 ax 的關係
  • Figure（fig）：整個畫布，可存成單一圖檔
  • Axes（ax）：一個繪圖區域（一張子圖），多數操作都在 ax 上
  • 一個 figure 可以有多個 axes

§27.2子圖與多軸

python 
# 2 列 2 行的子圖
fig, axes = plt.subplots(2, 2, figsize=(10, 8))

axes[0, 0].plot(x, np.sin(x))
axes[0, 0].set_title("sin")

axes[0, 1].plot(x, np.cos(x))
axes[0, 1].set_title("cos")

axes[1, 0].plot(x, x**2)
axes[1, 1].plot(x, np.exp(x))

# 共用 x 軸
fig, axes = plt.subplots(3, 1, sharex=True, figsize=(8, 8))

# 自動調整間距
fig.tight_layout()
# 或統一 super title
fig.suptitle("All functions", fontsize=16)

# 雙 y 軸
fig, ax1 = plt.subplots()
ax1.plot(x, y1, "b-")
ax1.set_ylabel("y1", color="blue")

ax2 = ax1.twinx()     # 共享 x 軸的第二個 y 軸
ax2.plot(x, y2, "r-")
ax2.set_ylabel("y2", color="red")

§27.3常見圖表類型

python 
fig, ax = plt.subplots()

# 折線圖
ax.plot(x, y, linestyle="--", marker="o", color="tab:blue")

# 散點圖
ax.scatter(x, y, s=50, c=colors, alpha=0.6, cmap="viridis")

# 長條圖
categories = ["A", "B", "C", "D"]
values = [23, 45, 56, 78]
ax.bar(categories, values, color="steelblue")
ax.barh(categories, values)      # 水平

# 直方圖
data = np.random.randn(1000)
ax.hist(data, bins=30, edgecolor="white")

# 箱形圖
ax.boxplot([data1, data2, data3], labels=["A", "B", "C"])

# 熱力圖
matrix = np.random.rand(10, 12)
im = ax.imshow(matrix, cmap="RdYlBu_r", aspect="auto")
fig.colorbar(im, ax=ax)

# 填充區域
ax.fill_between(x, y_low, y_high, alpha=0.3)      # 信賴區間

# 圓餅圖（盡量避免，長條圖更好看）
ax.pie(values, labels=categories, autopct="%1.1f%%")

# 加文字、箭頭
ax.text(5, 0.5, "重要點", fontsize=12)
ax.annotate("最高點", xy=(peak_x, peak_y),
            xytext=(peak_x+1, peak_y+1),
            arrowprops={"arrowstyle": "->"})

§27.4樣式與美化

python 
# 內建樣式
print(plt.style.available)       # 所有樣式
plt.style.use("seaborn-v0_8")       # 推薦
plt.style.use("ggplot")
plt.style.use("fivethirtyeight")

# 顏色指定
ax.plot(x, y, color="#3b82f6")      # hex
ax.plot(x, y, color="tab:blue")     # 預設調色盤
ax.plot(x, y, color=(0.2, 0.4, 0.8))   # RGB 0-1

# 中文字型（macOS / Windows / Linux 各自設定）
plt.rcParams["font.sans-serif"] = ["Microsoft JhengHei", "Heiti TC", "Noto Sans TC"]
plt.rcParams["axes.unicode_minus"] = False

# 全域預設（調一次即可）
plt.rcParams.update({
    "figure.figsize": (8, 5),
    "font.size": 12,
    "axes.grid": True,
    "grid.alpha": 0.3,
    "axes.spines.top": False,     # 去掉上邊框
    "axes.spines.right": False,
})

# 改刻度
ax.set_xticks([0, 2, 4, 6])
ax.set_xticklabels(["Jan", "Feb", "Mar", "Apr"], rotation=45)

# 設座標範圍
ax.set_xlim(0, 10)
ax.set_ylim(-1, 1)

# 對數軸
ax.set_yscale("log")
💡 好圖表的原則
  • 去除「墨水噪音」—— 不必要的邊框、網格、裝飾
  • 標籤清楚完整（標題、軸名、單位、圖例）
  • 色盲友善配色（viridis、cividis 調色盤）
  • 儲存用向量格式（PDF / SVG）給論文或印刷
  • 儲存用 PNG（dpi ≥ 150）給網頁

§27.5seaborn —— 統計圖專家

seaborn 是 matplotlib 的高階包裝，專注「看出資料間的關係」—— 相關性、分佈、組間差異。

bash 
uv add seaborn
python 
import seaborn as sns
import pandas as pd

sns.set_theme(style="whitegrid")       # 整體樣式

# 載入內建資料集測試
tips = sns.load_dataset("tips")

# 1. 散點圖 + 迴歸線
sns.regplot(data=tips, x="total_bill", y="tip")

# 2. 依類別著色
sns.scatterplot(data=tips, x="total_bill", y="tip",
                hue="time", style="sex", size="size")

# 3. 分佈
sns.histplot(data=tips, x="total_bill", kde=True)
sns.kdeplot(data=tips, x="total_bill", hue="time", fill=True)

# 4. 分類比較
sns.boxplot(data=tips, x="day", y="total_bill", hue="time")
sns.violinplot(data=tips, x="day", y="tip")
sns.barplot(data=tips, x="day", y="total_bill", errorbar="sd")

# 5. 關係矩陣（pair plot）
sns.pairplot(tips, hue="time")

# 6. 熱力圖（相關矩陣）
corr = tips.select_dtypes("number").corr()
sns.heatmap(corr, annot=True, cmap="RdYlBu_r", center=0)

# 7. Facet grid — 依類別切分子圖
sns.FacetGrid(tips, col="time", row="sex").map(
    sns.histplot, "total_bill"
)

# 8. 組合圖（jointplot）
sns.jointplot(data=tips, x="total_bill", y="tip", kind="hex")

§27.6plotly —— 互動式圖表

plotly 產出可以在瀏覽器中互動的 HTML 圖表：hover 顯示數值、縮放、篩選圖例，適合儀表板與網頁內嵌。

bash 
uv add plotly
python 
import plotly.express as px
import plotly.graph_objects as go

# plotly.express — 高階 API（類似 seaborn 風格）
df = px.data.gapminder()

# 散點圖
fig = px.scatter(df.query("year == 2007"),
                 x="gdpPercap", y="lifeExp",
                 size="pop", color="continent",
                 hover_name="country",
                 log_x=True, size_max=60,
                 title="Life Expectancy vs GDP (2007)")
fig.show()

# 折線圖（時間序列）
fig = px.line(df, x="year", y="lifeExp",
              color="country", line_group="continent")

# 長條圖（可動畫）
fig = px.bar(df, x="continent", y="pop",
             color="country", animation_frame="year")

# 地圖（等級色圖）
fig = px.choropleth(df.query("year == 2007"),
                    locations="iso_alpha",
                    color="lifeExp",
                    hover_name="country",
                    color_continuous_scale="Plasma")

# 3D 散點
fig = px.scatter_3d(df, x="gdpPercap", y="lifeExp", z="pop",
                    color="continent")

# 匯出
fig.write_html("chart.html")           # 互動式 HTML
fig.write_image("chart.png", scale=2)   # 需安裝 kaleido

# Jupyter 中直接顯示
fig.show()

低階 API（graph_objects）

python 
# 需要精細控制時用 go
fig = go.Figure()
fig.add_trace(go.Scatter(x=x, y=y1, name="Line 1"))
fig.add_trace(go.Scatter(x=x, y=y2, name="Line 2", line={"dash": "dash"}))

fig.update_layout(
    title="My Chart",
    xaxis_title="X",
    yaxis_title="Y",
    hovermode="x unified",      # 統一 hover 提示
    template="plotly_white",
)
💡 Dash 打造儀表板

plotly 的兄弟產品 Dash（uv add dash）讓你用 Python 寫完整的互動式儀表板應用，適合內部資料工具。

§27.7DataFrame 內建 plot

pandas 與 Polars 都有 .plot()（底層用 matplotlib / hvplot），最適合探索期間快速看看。

python 
# pandas 內建 plot
df["price"].plot()                         # 折線圖
df["price"].plot.hist(bins=30)              # 直方圖
df.plot.scatter(x="age", y="salary")
df.plot.bar(x="city", y="count")
df.plot.box()                             # 每個數值欄的箱形圖

# 多欄同時畫
df[["open", "close", "high"]].plot()

# 搭配 groupby
df.groupby("city")["salary"].mean().plot.bar()

# Polars 用 hvplot 或 altair 做互動
# 或先 .to_pandas().plot() 最快
df.to_pandas().plot()
