CHAPTER 05 · PART 2

流程控制與迴圈

條件判斷與迴圈是所有程式語言的骨幹。Python 的語法設計將這些結構打磨得極其優雅 —— 特別是一些獨特特性（for/while 的 else、zip、enumerate），讓熟練的 Python 使用者寫出明顯更 Pythonic 的程式碼。

Python 3.14 控制流結構 預估閱讀 18 分鐘

§0章節導論

本章從最基本的 if/else 開始，延伸到 Python 獨有的迴圈特性。重點：

  • 條件判斷的三種形態：傳統 if、三元表達式、短路邏輯
  • for 迴圈的真實意義（迭代器協議，不只是「數數」）
  • break、continue 與少見但強大的 else 子句
  • 迴圈進階工具：enumerate、zip、reversed、sorted

下一章將介紹 Python 3.10 引入的 結構化模式匹配（match/case），那是另一種強大的控制流機制。

§5.1if / elif / else

Python 的條件判斷語法極簡，沒有括號、沒有 {}、沒有 then。

python 
# 最基本：if - elif - else
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

# 只有 if，沒有 else — 完全合法
if score >= 60:
    print("及格！")

善用 Truthy / Falsy 簡化判斷

Python 的任何物件都可直接放進 if 判斷，不需要每次都寫 == 0、!= None 等冗長比較。

python 
names = ["Alice", "Bob"]

# ❌ 過度冗長
if len(names) != 0:
    print("有人")

# ✅ Pythonic — 空 list 即 falsy
if names:
    print("有人")

# 檢查 None 用 is / is not
result = None
if result is None:
    print("沒結果")

# 常見組合：檢查 dict 有 key 且值非空
if config.get("api_key"):
    connect(config["api_key"])
📘 == None 還是 is None？

一律用 is None / is not None。== 可能被 __eq__ 覆寫產生意外行為，而 is 比較的是身份（identity），永遠可靠且更快。PEP 8 也明文規定此一慣例。

§5.2三元條件表達式

Python 的三元語法順序與 C 系語言不同，採用「值 if 條件 else 值」的自然語序。

python 
# 語法：a if condition else b
age = 20
status = "adult" if age >= 18 else "minor"

# 用於推導式
labels = ["odd" if x%2 else "even" for x in range(5)]

# 設預設值（取代 None 時）
value = config.get("debug")
debug = value if value is not None else False

# 搭配短路 or（當「falsy 即取預設」時更簡潔）
name = user_input or "guest"        # 空字串/None 都會變 "guest"

§5.3for 迴圈

Python 的 for 本質上是「對可迭代物件逐一取值」，不像 C/Java 那樣是計數器。這點是理解 Python 控制流的關鍵。

python 
# 走訪 list
for fruit in ["apple", "banana", "cherry"]:
    print(fruit)

# 走訪字串（逐字元）
for ch in "Python":
    print(ch)

# 走訪 dict（預設為 key）
user = {"name": "Alice", "age": 30}
for key in user:
    print(key, user[key])

# 走訪 dict 的 (key, value) 對
for k, v in user.items():
    print(f"{k} = {v}")

# 走訪固定次數 — 用 range
for i in range(5):           # 0, 1, 2, 3, 4
    print(i)

for i in range(2, 10, 2):    # 2, 4, 6, 8（start, stop, step）
    print(i)

for i in range(10, 0, -1):   # 倒數 10 → 1
    print(i)

巢狀迴圈

python 
# 九九乘法表
for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i}×{j}={i*j:2d}", end=" ")
    print()  # 每列後換行

# 走訪二維 list
matrix = [[1, 2, 3], [4, 5, 6]]
for row in matrix:
    for val in row:
        print(val, end=" ")
💡 Pythonic 的 for 迴圈

不要這樣寫：for i in range(len(lst)): print(lst[i]) —— 這是 C 思維。
直接寫：for item in lst: print(item)；需要索引時用 enumerate（§5.7）。

§5.4while 迴圈

當「不知道要跑幾次」時用 while。條件為 True 時持續執行。

python 
# 基本形式
n = 0
while n < 5:
    print(n)
    n += 1

# 等待條件成立（持續輸入驗證）
while True:
    user = input("輸入數字：")
    if user.isdigit():
        break
    print("請重新輸入")

# Walrus := 讓 while 更簡潔（3.8+）
while (line := f.readline()):
    process(line)    # 讀到空字串（EOF）自動停止
⚠️ 無窮迴圈陷阱

忘記更新條件變數是新手最常見的錯誤，會讓程式卡死。while True: 必須搭配 break 才能退出。寫迴圈時養成先想退出條件的習慣。

§5.5break 與 continue

python 
# break — 完全跳出最內層迴圈
for i in range(100):
    if i == 5:
        break         # 遇到 5 就停
    print(i)
# 輸出：0 1 2 3 4

# continue — 跳過本次迴圈剩餘部分，進入下一輪
for i in range(10):
    if i % 2:
        continue      # 奇數跳過
    print(i)
# 輸出：0 2 4 6 8

# 巢狀迴圈中，break/continue 只影響最內層
for i in range(3):
    for j in range(3):
        if j == 2:
            break     # 只跳出 j 迴圈
        print(i, j)

跳出多層迴圈的技巧

Python 沒有 break 2 或標籤語法，想跳出多層迴圈常見手法：

python 
# 手法 1：用 flag 變數
found = False
for i in range(10):
    for j in range(10):
        if target(i, j):
            found = True
            break
    if found:
        break

# 手法 2：把多層迴圈抽成函式，用 return 提前返回（最推薦）
def find_target():
    for i in range(10):
        for j in range(10):
            if target(i, j):
                return i, j
    return None

# 手法 3：善用 itertools.product（扁平化為單層）
from itertools import product
for i, j in product(range(10), range(10)):
    if target(i, j):
        break      # 一個 break 就能跳出

§5.6for / while 的 else 子句（Python 獨有）

這是 Python 中很多人不熟悉的特性：for 和 while 可以接 else，在迴圈正常結束（未被 break）時執行。

python 
# 經典應用：搜尋與「找不到」的處理
for item in inventory:
    if item.id == target_id:
        print("找到了！")
        break
else:
    # 迴圈跑完沒被 break → 沒找到
    print("找不到")

# 檢查質數
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False    # 找到因數
    else:
        return True         # 沒找到因數 → 是質數
📘 記憶訣竅

把 for...else 的 else 想成 "no break" 會比較自然 —— 「如果迴圈沒有被 break 打斷，就執行 else」。很多 Python 設計者也曾說這個關鍵字名稱選得不夠好。

§5.7range、enumerate、zip

range — 產生整數序列

python 
range(5)           # 0, 1, 2, 3, 4
range(2, 8)        # 2, 3, 4, 5, 6, 7
range(0, 10, 2)    # 0, 2, 4, 6, 8
range(10, 0, -1)   # 10, 9, 8 ... 1

# range 是 lazy 的，不會立刻產生所有值
for i in range(10**9):   # 不會爆記憶體
    if i > 100:
        break

# 需要 list 時手動轉
list(range(5))     # [0, 1, 2, 3, 4]

enumerate — 同時取得索引與值

python 
fruits = ["apple", "banana", "cherry"]

# ❌ 不 Pythonic
for i in range(len(fruits)):
    print(i, fruits[i])

# ✅ Pythonic — enumerate
for i, fruit in enumerate(fruits):
    print(i, fruit)

# 指定起始索引
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")    # 從 1 開始

zip — 多序列並行走訪

python 
names = ["Alice", "Bob", "Charlie"]
ages = [30, 25, 35]

for name, age in zip(names, ages):
    print(f"{name}: {age}")

# 三個以上也可以
cities = ["Taipei", "Tokyo", "Seoul"]
for n, a, c in zip(names, ages, cities):
    print(n, a, c)

# 長度不同時，以最短的為準
list(zip([1, 2, 3], ["a", "b"]))   # [(1,'a'), (2,'b')]

# strict=True 要求等長，否則 ValueError（3.10+）
list(zip([1, 2, 3], ["a", "b"], strict=True)) # 🚫 ValueError

# 把 list of tuples 拆解（zip 反向操作）
pairs = [(1, "a"), (2, "b"), (3, "c")]
nums, letters = zip(*pairs)          # (1,2,3), ('a','b','c')

§5.8pass — 空區塊的佔位

Python 語法要求區塊至少有一行陳述句。當你還沒想好實作時，用 pass 佔位。

python 
def todo_later():
    pass     # 之後再實作

class EmptyClass:
    pass

if x < 0:
    pass     # 明確表達「這種情況故意不處理」
else:
    process(x)

# 也可以用 ... (Ellipsis) 替代
def future_impl():
    ...

§5.9迴圈進階工具

sorted / reversed

python 
# sorted — 產生新序列（不改原序列）
for n in sorted([3, 1, 4, 1, 5]):
    print(n)    # 1 1 3 4 5

# 按 key 排序
names = ["Charlie", "alice", "BOB"]
sorted(names, key=str.lower)    # ['alice', 'BOB', 'Charlie']

sorted(users, key=lambda u: u.age, reverse=True)

# 多重條件排序
sorted(users, key=lambda u: (u.age, u.name))

# reversed — 反向迭代
for n in reversed([1, 2, 3]):
    print(n)    # 3 2 1

any / all — 迴圈的速記

python 
nums = [1, 2, 3, 4]

any(n > 3 for n in nums)    # True（至少一個符合）
all(n > 0 for n in nums)    # True（全部符合）
all(nums)                    # True（全部 truthy）
any(nums)                    # True（至少一個 truthy）

# 驚喜：空序列
all([])     # True（vacuous truth）
any([])     # False

map / filter — 函數式工具

python 
# map — 對每個元素套函式
list(map(str.upper, ["a", "b", "c"]))    # ['A', 'B', 'C']
list(map(lambda x: x**2, range(5)))  # [0,1,4,9,16]

# filter — 過濾
list(filter(lambda x: x>0, [-1, 2, -3, 4])) # [2, 4]

# ⚠️ 多數情況下，推導式更 Pythonic
[x**2 for x in range(5)]
[x for x in nums if x > 0]
