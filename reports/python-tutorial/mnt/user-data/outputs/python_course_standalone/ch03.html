字串是人與程式的橋樑。Python 在 3.14 引入了全新的 t-string（Template String），補足了 f-string 在安全性上的缺口 —— 讓 SQL Injection、XSS 攻擊從根本被消除。本章帶你從基礎一路走到這項 2026 年最重要的新特性。

Python 3.14 包含 PEP 750 t-string 預估閱讀 25 分鐘

§3.1字串基礎

Python 的字串是不可變（immutable）的 Unicode 字元序列。可用多種引號定界：

python 
s1 = '單引號'
s2 = "雙引號"               # 兩者等價，選一種統一使用
s3 = '''三引號
可跨越多行'''
s4 = """也可以用雙三引號
常用於 docstring"""

# Raw string — 反斜線不被解釋為跳脫
path = r"C:\Users\Alice\Documents"   # 不用寫 C:\\Users...
regex = r"\d+\.\d+"                     # 寫 regex 必備

# 跳脫字元
print("Hello\tWorld")       # Tab
print("Line1\nLine2")       # 換行
print("She said \"Hi\"")    # 轉義雙引號
print("\u4f60\u597d")        # Unicode 跳脫：你好
print("\N{GREEK SMALL LETTER PI}")  # π

# 字串串接與重複
s = "Hello" + " " + "World"  # Hello World
banner = "-" * 40                # 40 個減號

# 相鄰字串字面值自動串接（重要技巧）
long_msg = (
    "這是一段"
    "非常長的"
    "訊息。"
)   # 自動合併為「這是一段非常長的訊息。」

字串是不可變的

python 
s = "Hello"
s[0] = "h"       # ❌ TypeError！str 不支援賦值

# 要改就產生新字串
s = "h" + s[1:]   # 'hello'

§3.2索引與切片

python 
s = "Python"
#   0  1  2  3  4  5
#  -6 -5 -4 -3 -2 -1

s[0]       # 'P'
s[-1]      # 'n'（倒數第一）
s[-2]      # 'o'

# 切片 [start : stop : step] — 包含 start，不含 stop
s[0:3]     # 'Pyt'
s[2:]      # 'thon'（從 2 到結尾）
s[:4]      # 'Pyth'（從開頭到 4）
s[:]       # 'Python'（完整複製）
s[::2]     # 'Pto'（每兩個取一個）
s[::-1]    # 'nohtyP'（反轉字串）
s[1:5:2]   # 'yh'

# 越界不會錯（與 list[index] 不同）
s[100:]    # ''（空字串，不報錯）

§3.3常用字串方法

大小寫與判斷

python 
"hello".upper()       # 'HELLO'
"HELLO".lower()       # 'hello'
"hello world".title()     # 'Hello World'
"Hello".swapcase()    # 'hELLO'
"hello".capitalize()  # 'Hello'

"abc123".isalnum()   # True（字母或數字）
"abc".isalpha()      # True（純字母）
"123".isdigit()      # True
"  ".isspace()       # True
"Hello".istitle()    # True
"abc".islower()      # True
"ABC".isupper()      # True

搜尋與取代

python 
s = "Hello, World!"

"World" in s          # True
s.startswith("Hello")  # True
s.endswith("!")       # True
s.startswith(("Hi", "Hello"))  # True（tuple 任一符合）

s.find("o")         # 4（第一個位置）
s.rfind("o")        # 8（最後一個）
s.find("z")         # -1（找不到）
s.index("o")        # 4（找不到會 ValueError）
s.count("l")        # 3

s.replace("World", "Python")  # 'Hello, Python!'
s.replace("l", "L", 1)       # 只替換第一個

切割與合併

python 
"a,b,c".split(",")         # ['a', 'b', 'c']
"a b c".split()            # ['a', 'b', 'c']（預設空白）
"a,b,c,d".split(",", 2)    # ['a', 'b', 'c,d']（最多切 2 次）
"a\nb\nc".splitlines()     # ['a', 'b', 'c']
"a,b,c".partition(",")     # ('a', ',', 'b,c')（只切第一次）

"-".join(["a", "b", "c"])  # 'a-b-c'
"".join(["H", "i"])       # 'Hi'

去除空白與填充

python 
"   hello   ".strip()      # 'hello'
"   hello   ".lstrip()     # 'hello   '
"   hello   ".rstrip()     # '   hello'
"***hello***".strip("*")   # 'hello'（指定字元）

# 移除前後綴（3.9+）
"HelloWorld".removeprefix("Hello")  # 'World'
"test.py".removesuffix(".py")     # 'test'

# 填充與對齊
"5".zfill(3)         # '005'
"hi".ljust(10, ".")   # 'hi........'
"hi".rjust(10, ".")   # '........hi'
"hi".center(10, "*")  # '****hi****'

§3.4f-string 進階用法（最常用）

f-string（Python 3.6+）是現代 Python 的主流字串格式化方式 —— 直觀、快速、強大。

python 
name = "Alice"
age = 30
price = 1234.5678

# 基本插值
print(f"{name} is {age} years old")

# 運算式
print(f"{age + 5} years from now")
print(f"{name.upper()}")

# 格式化規格：{value:spec}
print(f"{price:.2f}")          # 1234.57（2 位小數）
print(f"{price:,.2f}")         # 1,234.57（千分位）
print(f"{price:10.2f}")        # '   1234.57'（寬度 10）
print(f"{price:<10.2f}")       # 左對齊
print(f"{price:>10.2f}")       # 右對齊（預設）
print(f"{price:^10.2f}")       # 置中
print(f"{price:*^10.2f}")      # 用 * 填充置中

print(f"{255:b}")            # '11111111'（二進位）
print(f"{255:o}")            # '377'（八進位）
print(f"{255:x}")            # 'ff'（十六進位）
print(f"{255:#x}")           # '0xff'
print(f"{0.12345:%}")         # '12.345000%'
print(f"{0.12345:.1%}")       # '12.3%'

# 除錯用 = （3.8+ 自除錯）
print(f"{name=}, {age=}")     # name='Alice', age=30

# 日期格式化
from datetime import datetime
now = datetime.now()
print(f"{now:%Y-%m-%d %H:%M:%S}")

# 巢狀（寬度用變數）
width = 10
print(f"{price:{width}.2f}")

# 多行 f-string（3.12+ 允許更自由的語法）
msg = f"""
Name: {name}
Age:  {age}
"""

轉換旗標 !r !s !a

python 
x = "hello"
print(f"{x}")      # hello
print(f"{x!r}")     # 'hello'（呼叫 repr()）
print(f"{x!s}")     # hello （呼叫 str()）
print(f"{x!a}")     # 'hello'（呼叫 ascii()）

§3.5t-string：Template String（3.14 新特性）

🆕 Python 3.14 PEP 750 — t-string

t-string 是 f-string 的一般化：語法幾乎相同，只是改用 t"..." 前綴。關鍵差別：f-string 立刻變成 str，t-string 則會產生一個 Template 物件，讓你在「字串組合之前」攔截並處理變數 —— 這是防止 SQL injection、XSS、log injection 的語言級解方。

基本語法

python · Python 3.14+ 
name = "Pythonista"
day = "Friday"

# f-string 立刻變成 str
f_result = f"Hello, {name}! Today is {day}."
print(type(f_result))
# <class 'str'>

# t-string 產生 Template 物件
t_result = t"Hello, {name}! Today is {day}."
print(type(t_result))
# <class 'string.templatelib.Template'>

# Template 物件保留了「結構」
print(t_result.strings)
# ('Hello, ', '! Today is ', '.')

print(t_result.values)
# ('Pythonista', 'Friday')

print(t_result.interpolations)
# (Interpolation('Pythonista', 'name', None, ''),
#  Interpolation('Friday', 'day', None, ''))

為什麼需要 t-string？—— 安全性

python · 危險！ 
# ❌ 用 f-string 組 SQL：經典 SQL Injection 漏洞
user_input = "'; DROP TABLE users; --"
sql = f"SELECT * FROM users WHERE name = '{user_input}'"
cursor.execute(sql)   # 💀 整個 users 表被刪

# ❌ 用 f-string 組 HTML：XSS 漏洞
user = "<script>alert('pwn')</script>"
html = f"<p>{user}</p>"   # 💀 惡意腳本被注入
python · 用 t-string 解決 
from string.templatelib import Template, Interpolation

# 寫一個 HTML 跳脫處理器
def html(tmpl: Template) -> str:
    import html as html_mod
    parts = []
    for item in tmpl:
        if isinstance(item, Interpolation):
            parts.append(html_mod.escape(str(item.value)))
        else:
            parts.append(item)
    return "".join(parts)

# ✅ 攻擊字串被安全地跳脫
user = "<script>alert('pwn')</script>"
safe = html(t"<p>{user}</p>")
print(safe)
# <p>&lt;script&gt;alert('pwn')&lt;/script&gt;</p>

Interpolation 物件的完整結構

python 
price = 1234.5678
tmpl = t"Balance: {price!s:.2f}"

# 逐一檢視
for item in tmpl:
    print(item)
# 'Balance: '
# Interpolation(
#     value=1234.5678,          ← 運算後的值
#     expression='price',       ← 原始表達式
#     conversion='s',           ← !r !s !a 轉換
#     format_spec='.2f'         ← 格式規格
# )

實用範例：安全的 logging

python 
from string.templatelib import Template, Interpolation
import json

def log_structured(tmpl: Template) -> dict:
    """將 t-string 轉為結構化 log（JSON-ready）"""
    message_parts = []
    fields = {}
    for item in tmpl:
        if isinstance(item, Interpolation):
            message_parts.append(str(item.value))
            fields[item.expression] = item.value
        else:
            message_parts.append(item)
    return {
        "message": "".join(message_parts),
        "fields": fields
    }

action, amount, item = "traded", 42, "shrubs"
log = log_structured(t"User {action}: {amount} {item}")
print(json.dumps(log, indent=2))
# {
#   "message": "User traded: 42 shrubs",
#   "fields": {"action": "traded", "amount": 42, "item": "shrubs"}
# }

§3.6其他格式化：str.format 與 %

為了相容舊專案、讀懂他人程式碼，仍需了解這兩種格式化方式。

python 
# str.format — Python 2.6+ / 3.0+
"{} is {} years old".format("Alice", 30)
"{0} and {1}, {0}".format("a", "b")    # 'a and b, a'
"{name}: {age}".format(name="A", age=5)
"{:.2f}".format(3.14159)              # '3.14'

# % 格式化（C 風格，最古老，logging 模組仍廣泛使用）
"%s is %d years old" % ("Alice", 30)
"%(name)s: %(age)d" % {"name": "A", "age": 5}
"%.2f" % 3.14159                       # '3.14'

§3.7編碼與 bytes

Python 3 的 str 是 Unicode 字串，bytes 是原始位元組序列 —— 兩者必須明確轉換。

python 
# str → bytes (encode)
s = "你好"
b = s.encode("utf-8")     # b'\xe4\xbd\xa0\xe5\xa5\xbd'
b = s.encode()              # 預設 utf-8

# bytes → str (decode)
s2 = b.decode("utf-8")      # '你好'

# bytes 字面值
data = b"hello"             # 只能 ASCII
data = bytes([65, 66, 67])    # b'ABC'

# 常見編碼
# utf-8 / utf-16 / ascii / latin-1 / big5 / gb2312 / gbk

# 處理錯誤
s.encode("ascii", errors="ignore")    # 忽略不能編碼的字元
s.encode("ascii", errors="replace")   # 替換為 ?
s.encode("ascii", errors="xmlcharrefreplace")
⚠️ 檔案讀寫要指定 encoding
# ❌ 在 Windows 上會用 cp950 開啟，常導致亂碼
open("file.txt")

# ✅ 永遠指定 encoding
open("file.txt", encoding="utf-8")

§3.8正規表達式（re 模組）

python 
import re

text = "email: alice@example.com, bob@test.org"

# 搜尋第一個符合
m = re.search(r"\w+@\w+\.\w+", text)
if m:
    print(m.group())        # alice@example.com
    print(m.start(), m.end())   # 位置

# 找全部
emails = re.findall(r"\w+@\w+\.\w+", text)
# ['alice@example.com', 'bob@test.org']

# 分組 groups()
m = re.search(r"(\w+)@(\w+\.\w+)", text)
print(m.groups())            # ('alice', 'example.com')
print(m.group(1))            # 'alice'

# 具名分組
m = re.search(r"(?P<user>\w+)@(?P<domain>\w+\.\w+)", text)
print(m.group("user"))      # 'alice'

# 取代
re.sub(r"\d+", "***", "aa 123 bb 456")
# 'aa *** bb ***'

# 切割
re.split(r"[,;\s]+", "a, b; c  d")
# ['a', 'b', 'c', 'd']

# 編譯（重複使用時更快）
pattern = re.compile(r"\d+")
pattern.findall("a1 b22 c333")   # ['1', '22', '333']
