基礎語法與資料型別

Python 的語法乾淨到近乎詩意。本章帶你掌握變數、縮排規則、數值運算、布林邏輯等基石 —— 它們簡單，但每一條規則背後都藏著 Python 的設計哲學：Readability counts。

Python 3.14 核心語法 預估閱讀 20 分鐘

§0章節導論

Python 之所以成為全球最受歡迎的程式語言之一，核心原因就是它的語法設計 ——「動態型別、強制縮排、萬物皆物件」三大原則讓新手友善，老手高效。本章內容：

  • 語法基本結構、陳述句（statement）與運算式（expression）的分別
  • Python 的命名規則與 PEP 8 風格指南
  • 數值型別的完整家族：int、float、complex、Decimal、Fraction
  • 運算子優先順序與常見陷阱

§2.1語法基石

Python 的程式由一連串陳述句（statements）組成，每個陳述句通常占一行。行尾不需要分號（;），但可以用來在同一行放多個陳述句（不建議）。

python 
# 單一陳述句一行（推薦）
x = 10
y = 20
print(x + y)

# 多陳述句一行（合法但不推薦）
x = 10; y = 20; print(x + y)

# 一行太長時用反斜線續行
total = 1 + 2 + 3 + \
        4 + 5 + 6

# 括號內的換行不需要反斜線（推薦）
total = (1 + 2 + 3
         + 4 + 5 + 6)
📘 Statement vs Expression

Expression（運算式）會產生一個值，如 2 + 3、len("abc")。
Statement（陳述句）執行一個動作，如 x = 10、if y > 0: ...。所有 expression 都可以當作 statement，但反之不行。

§2.2變數與命名規則

Python 是動態型別語言：變數不需要宣告型別，賦值當下自動決定。變數本質上是「指向物件的名稱」。

python 
age = 25              # int
name = "Alice"        # str
pi = 3.14159         # float
is_valid = True      # bool

# 多重賦值
a, b, c = 1, 2, 3
x = y = z = 0        # 鏈式賦值

# 交換變數（不需暫存變數！）
a, b = b, a

# 型別可隨時改變（動態型別）
x = 10             # int
x = "hello"        # 現在是 str，合法！

命名規則

  • 只能包含字母、數字、底線，且不能以數字開頭
  • 區分大小寫：name 和 Name 是兩個不同變數
  • 不能使用 Python 保留字（if、for、class、def 等 35 個）
  • 支援 Unicode：年齡 = 25 合法（但不建議）

PEP 8 命名慣例

§2.3註解

python 
# 這是單行註解

x = 42  # 行尾註解

"""
這是多行字串，常被拿來當作多行註解。
但嚴格來說它只是一個字串物件，
只有在當作函式/類別 docstring 時會被 Python 保留。
"""

def area(r: float) -> float:
    """計算圓面積。

    Args:
        r: 半徑
    Returns:
        面積值
    """   # ← 這是 docstring，可被 help() 讀取
    return 3.14159 * r ** 2

§2.4縮排規則

Python 用縮排表達程式區塊（block），取代 C/Java 的 {}。這讓程式碼視覺上自然對齊，但也要求嚴格一致。

python 
if x > 0:
    print("正數")       # 縮排 4 空格（PEP 8 標準）
    print("大於零")
else:
    print("非正數")

# ❌ 錯誤：同一區塊縮排不一致
if x > 0:
    print("A")
   print("B")          # IndentationError！
⚠️ 縮排鐵律
  • PEP 8 推薦使用 4 個空格，不要混用 Tab 與空格
  • 同一區塊必須縮排一致（不一定要 4，但要統一）
  • VS Code / PyCharm 會自動把 Tab 轉成空格，可避免陷阱

§2.5數值型別

三大內建數值型別

python 
# int — 整數（任意精度，Python 不會溢位）
a = 42
big = 10 ** 100           # 10 的 100 次方，OK
hex_val = 0xFF             # 16 進位 = 255
oct_val = 0o17             # 8 進位 = 15
bin_val = 0b1010           # 2 進位 = 10
million = 1_000_000        # 底線分隔，更易讀

# float — 浮點數（IEEE 754 雙精度）
pi = 3.14159
speed_of_light = 2.998e8   # 科學記號 = 2.998 × 10⁸
inf_val = float('inf')      # 無窮大
nan_val = float('nan')      # Not a Number

# complex — 複數
z = 3 + 4j
print(z.real, z.imag)         # 3.0 4.0
print(abs(z))                 # 5.0（模長）

高精度數值：Decimal 與 Fraction

python 
# float 的陷阱：0.1 + 0.2 != 0.3
print(0.1 + 0.2)              # 0.30000000000000004

# 處理金錢用 Decimal（精確小數）
from decimal import Decimal
price = Decimal('0.1') + Decimal('0.2')
print(price)                  # 0.3（精確！）

# 處理分數用 Fraction
from fractions import Fraction
f = Fraction(1, 3) + Fraction(1, 6)
print(f)                      # 1/2
💡 何時使用哪種數值型別？
  • int：計數、索引、離散量
  • float：科學計算、距離、速度等（允許誤差）
  • Decimal：金融、會計、貨幣（必須精確）
  • Fraction：有理數運算、教學用
  • complex：訊號處理、電路分析、量子計算

§2.6布林（bool）與 None

python 
# bool 只有兩個值：True、False（首字母大寫）
is_active = True
is_deleted = False

# bool 實際上是 int 的子類別
print(True + True)             # 2
print(isinstance(True, int))   # True

# Truthy / Falsy：任何物件都可在布林情境使用
# Falsy 值：False, None, 0, 0.0, '', [], {}, set(), ()
if []:
    print("不會印出")
else:
    print("空 list 是 falsy")

# None — 代表「沒有值」（相當於其他語言的 null）
result = None

# 檢查 None 一律用 is / is not，不用 == / !=
if result is None:
    print("沒有結果")

比較運算子（回傳 bool）

運算子意義範例
==值相等1 == 1.0 # True
!=不相等"a" != "b"
<, >, <=, >=大小比較3 < 5
is, is not身份相等（同一物件）x is None
in, not in成員關係3 in [1,2,3]
python 
# Python 支援鏈式比較（數學記法）
if 0 < x < 100:
    print("在範圍內")   # 比 0 < x and x < 100 更優雅

# == vs is 的差別
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)   # True（內容相同）
print(a is b)   # False（不同物件）

邏輯運算子（短路求值）

python 
print(True and False)   # False
print(True or False)    # True
print(not True)          # False

# 短路：找到決定性結果就停止
print(0 and print("不會執行"))   # 印出 0
print(1 or print("不會執行"))    # 印出 1

# and / or 回傳的是運算元本身，不一定是 bool
print("hi" and "bye")     # 'bye'（兩者都 truthy 時回傳後者）
print(None or "default")  # 'default'（第一個 falsy 時回傳後者）

賦值運算子（含 Walrus :=）

python

x -= 3        # 12

# Walrus := — 賦值同時回傳值（Python 3.8+）
if (n := len(data)) > 10:

§2.8input 與 print

python 
# print — 輸出
print("Hello", "World")            # 預設用空格分隔：Hello World
print("A", "B", "C", sep="-")    # A-B-C
print("no newline", end="")      # 結尾不換行
print("err", file=sys.stderr)     # 輸出到 stderr

# input — 輸入（一律回傳 str）
name = input("你的名字？")
age = int(input("幾歲？"))        # 需要手動轉型
print(f"你好 {name}，你 {age} 歲")

§2.9型別轉換

python 
# 顯式轉型（強制）
int("42")         # 42
int(3.7)          # 3（截去小數）
int("ff", 16)     # 255（指定進位）

float("3.14")     # 3.14
float("1e-3")     # 0.001

str(42)           # '42'
str([1, 2])       # '[1, 2]'

bool(0)          # False
bool("false")    # True（非空字串都是 True！）

# 型別檢查
type(42)              # <class 'int'>
isinstance(42, int)   # True（推薦此方式）
isinstance(3.14, (int, float))  # True（多型檢查）
⚠️ 常見陷阱
  • int("3.14") 會 ValueError，要先 int(float("3.14"))
  • bool("False") 是 True（非空字串）— 解析設定檔要特別注意
  • float('inf') + 1 == float('inf')，nan != nan
