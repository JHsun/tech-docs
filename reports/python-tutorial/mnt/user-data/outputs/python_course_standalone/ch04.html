四大容器資料結構

Python 的四大內建容器 —— list、tuple、set、dict —— 構成了 80% 程式邏輯的骨架。理解它們的特性、適用時機，以及 Python 最優雅的武器：推導式，是跨越新手門檻的關鍵。

Python 3.14 內建資料結構 預估閱讀 25 分鐘

§0章節導論

「資料結構決定演算法」—— 選對容器，程式自然又快又好讀。本章涵蓋：

  • 四大容器的特性矩陣（有序性、可變性、重複性、雜湊）
  • 每種容器的常用操作、時間複雜度與陷阱
  • Python 最受喜愛的語法糖：推導式（comprehension）

§4.1四大容器總覽

容器語法有序可變允許重複元素需可雜湊典型用途

dict{"a": 1}✅(3.7+)✅key 唯一key 需可雜湊映射、查找表
📘 「可雜湊（hashable）」是什麼？

可雜湊物件有固定的 hash() 值，在生命週期內不變。不可變物件（int、str、tuple）預設可雜湊；可變物件（list、dict、set）不可雜湊，因此不能當 dict 的 key 或 set 的元素。

§4.2list — 有序可變序列

建立與基本操作

python 
# 建立 list
fruits = ["apple", "banana", "cherry"]
empty = []
from_range = list(range(5))        # [0, 1, 2, 3, 4]
from_str = list("abc")              # ['a', 'b', 'c']

# 索引與切片（與字串相同）
fruits[0]             # 'apple'
fruits[-1]            # 'cherry'
fruits[1:3]           # ['banana', 'cherry']
fruits[::-1]          # 反轉

# 修改（list 是可變的！）
fruits[0] = "avocado"
fruits[1:3] = ["b1", "b2", "b3"]   # 切片賦值，可改變長度

增刪方法

python 
lst = [1, 2, 3]

# 新增
lst.append(4)              # 尾端加入 → [1,2,3,4]
lst.extend([5, 6])         # 延展 → [1,2,3,4,5,6]
lst.insert(0, 0)           # 指定位置 → [0,1,2,3,4,5,6]
lst += [7, 8]              # += 等於 extend

# 刪除
lst.remove(3)              # 移除第一個符合的值（找不到會 ValueError）
last = lst.pop()            # 彈出尾端（也可 pop(index)）
del lst[0]                  # 刪除指定索引
del lst[1:3]                # 刪除切片
lst.clear()                  # 清空

# 查詢
lst.index(5)               # 回傳第一個 5 的位置
lst.count(5)               # 出現次數

# 排序與反轉
lst.sort()                   # 原地排序（升序）
lst.sort(reverse=True)       # 降序
lst.sort(key=str.lower)       # 指定排序鍵
sorted(lst)                  # 不改原 list，回傳新 list
lst.reverse()                # 原地反轉
reversed(lst)                # 回傳迭代器

⚠️ 淺拷貝與深拷貝

python 
a = [1, 2, 3]

# ❌ 這不是複製，是共享同一個 list！
b = a
b.append(4)
print(a)  # [1, 2, 3, 4] ← a 也變了

# ✅ 淺拷貝（shallow copy）
b = a.copy()        # 或 list(a)、a[:]

# ✅ 深拷貝（deep copy）— 巢狀結構需要
import copy
nested = [[1, 2], [3, 4]]
b = copy.deepcopy(nested)
⚠️ 建立二維串列的陷阱
# ❌ 所有 row 共享同一個 list！
grid = [[0] * 3] * 3
grid[0][0] = 1
print(grid)  # [[1,0,0], [1,0,0], [1,0,0]] 錯！

# ✅ 用推導式
grid = [[0] * 3 for _ in range(3)]

§4.3tuple — 不可變序列

tuple 幾乎是 list 的唯讀版本，但用途更深：固定記錄、多值回傳、可當 dict key。

python 
# 建立
point = (3, 4)
empty = ()
single = (42,)           # ⚠️ 逗號不可省！(42) 只是整數
no_paren = 1, 2, 3       # 括號可省略（tuple packing）

# 索引、切片、len、in — 與 list 相同
point[0]           # 3
len(point)         # 2
3 in point        # True

# 不可修改！
point[0] = 5       # ❌ TypeError

# 唯二的方法
point.count(3)      # 1
point.index(4)      # 1

# 多值回傳（最常見應用）
def min_max(lst):
    return min(lst), max(lst)   # 回傳 tuple

lo, hi = min_max([3, 1, 4, 1, 5])   # 解包接收

具名元組 NamedTuple

python 
from typing import NamedTuple

class Point(NamedTuple):
    x: float
    y: float

p = Point(3, 4)
print(p.x, p.y)       # 3 4（屬性存取）
print(p[0])           # 3（仍可索引）
print(p._asdict())     # {'x': 3, 'y': 4}

§4.4set — 無序不重複集合

python 
# 建立
s = {1, 2, 3}
empty = set()              # ⚠️ {} 是空 dict，不是空 set！
from_list = set([1, 2, 2, 3])  # {1, 2, 3} 自動去重
unique = set("abracadabra")     # {'a','b','r','c','d'}

# 增刪
s.add(4)
s.update([5, 6])         # 加入多個
s.remove(2)              # 找不到會 KeyError
s.discard(99)            # 找不到不會錯（安全）
s.pop()                   # 彈出任一元素
s.clear()

# 成員測試：O(1) 平均，比 list 快得多
3 in s                    # True

集合運算（數學集合）

python 
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b              # 聯集 {1,2,3,4,5,6}
a & b              # 交集 {3,4}
a - b              # 差集 {1,2}
a ^ b              # 對稱差 {1,2,5,6}

# 關係判斷
a.issubset(b)       # a ⊆ b ?
a.issuperset(b)     # a ⊇ b ?
a.isdisjoint(b)     # 不相交 ?

# frozenset — 不可變版本，可當 dict key
fs = frozenset([1, 2, 3])
fs.add(4)             # ❌ AttributeError
💡 set 最常見的兩個應用
  1. 快速去重：list(set(lst))（但會失序，若要保序用 list(dict.fromkeys(lst))）
  2. O(1) 成員測試：要頻繁判斷某元素是否存在時，把資料丟進 set 再查

§4.5dict — 鍵值對映射

python 
# 建立
d = {"name": "Alice", "age": 30}
empty = {}
d2 = dict(name="Alice", age=30)
d3 = dict([("a", 1), ("b", 2)])     # 從 tuple 序列
d4 = dict.fromkeys(["a", "b"], 0)     # {'a':0, 'b':0}

# 讀取
d["name"]              # 'Alice'（找不到會 KeyError）
d.get("email")         # None（安全：找不到回傳 None）
d.get("email", "N/A")  # 'N/A'（指定預設值）

# 新增 / 修改
d["email"] = "a@x.com"
d.update({"age": 31, "city": "Taipei"})

# 刪除
del d["age"]
email = d.pop("email")        # 移除並回傳
d.pop("missing", None)       # 有預設值不會錯
d.clear()

# 判斷
"name" in d             # True（檢查 key）

遍歷

python 
d = {"a": 1, "b": 2, "c": 3}

for k in d:                  # 預設迭代 key
    print(k)
for k in d.keys():           # 明確取 key
    pass
for v in d.values():         # 取 value
    pass
for k, v in d.items():      # 取 key + value（最常用）
    print(f"{k} = {v}")

合併字典

python 
a = {"x": 1, "y": 2}
b = {"y": 20, "z": 30}

# 3.9+ 合併運算子（最推薦）
merged = a | b            # {'x':1, 'y':20, 'z':30}（b 覆蓋 a）
a |= b                    # 原地合併

# 傳統方式
merged = {**a, **b}
a.update(b)

setdefault 與 defaultdict

python 
# setdefault：key 不存在就建立
d = {}
d.setdefault("items", []).append("x")
d.setdefault("items", []).append("y")
print(d)   # {'items': ['x', 'y']}

# defaultdict：更方便的分組利器
from collections import defaultdict

groups = defaultdict(list)
for word in ["apple", "ant", "banana", "bee"]:
    groups[word[0]].append(word)
print(dict(groups))
# {'a': ['apple', 'ant'], 'b': ['banana', 'bee']}

# Counter — 頻率統計的終極武器
from collections import Counter
c = Counter("mississippi")
print(c)                  # Counter({'i':4, 's':4, 'p':2, 'm':1})
print(c.most_common(2))   # [('i', 4), ('s', 4)]
📘 dict 自 Python 3.7 起保證插入順序

自 Python 3.7 起，dict 保證按插入順序迭代。若需要從頭/尾彈出（FIFO/LIFO），用 collections.OrderedDict，它有額外的 .move_to_end() 方法。

§4.6推導式 — Python 最優雅的語法糖

推導式（comprehension）讓你用一行寫出「對某序列做轉換/過濾」的邏輯，是 Pythonic 程式碼的標誌。

list 推導式

python 
# 基本：[運算式 for 變數 in 可迭代物件]
squares = [x**2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 加條件
evens = [x for x in range(20) if x % 2 == 0]

# 加 if-else 要放在前面
parity = ["odd" if x%2 else "even" for x in range(5)]

# 巢狀迴圈 — 類似巢狀 for
pairs = [(x, y) for x in [1, 2] for y in ["a", "b"]]
# [(1,'a'), (1,'b'), (2,'a'), (2,'b')]

# 扁平化二維串列
matrix = [[1, 2, 3], [4, 5, 6]]
flat = [n for row in matrix for n in row]   # [1,2,3,4,5,6]

dict / set 推導式

python 
# dict 推導式
squares = {x: x**2 for x in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

# 反轉 dict 的 key 與 value
d = {"a": 1, "b": 2}
inv = {v: k for k, v in d.items()}

# 過濾 dict
filtered = {k: v for k, v in d.items() if v > 1}

# set 推導式
unique_lengths = {len(w) for w in ["hi", "hello", "hey"]}
# {2, 3, 5}

生成器運算式（節省記憶體）

python 
# 小括號 → 生成器（lazy，一次產生一個值）
gen = (x**2 for x in range(10**8))   # 不會佔用大量記憶體

sum(x**2 for x in range(100))    # 不需額外括號
max(len(w) for w in words)

§4.7解包技巧（unpacking）

python 
# 基本解包
a, b, c = [1, 2, 3]
x, y = (10, 20)

# 星號解包：收集剩餘元素
first, *rest = [1, 2, 3, 4]        # first=1, rest=[2,3,4]
*init, last = [1, 2, 3, 4]          # init=[1,2,3], last=4
first, *mid, last = [1, 2, 3, 4, 5]  # first=1, mid=[2,3,4], last=5

# 交換變數
a, b = b, a

# 忽略不需要的值（慣例用 _）
_, name, _, age = ("id001", "Alice", "admin", 30)

# 函式引數解包
def add(x, y, z): return x + y + z
args = [1, 2, 3]
add(*args)                           # 展開為 add(1, 2, 3)

# 字典解包
kwargs = {"x": 1, "y": 2, "z": 3}
add(**kwargs)                        # 展開為 add(x=1, y=2, z=3)

# 合併容器
combined_list = [*lst1, *lst2]
combined_dict = {**d1, **d2}
