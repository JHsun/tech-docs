Data Flow — 三種執行路徑
⌨️ CLI Session
01使用者輸入
→HermesCLI.process_input()
→AIAgent.run_conversation()
→build_system_prompt()
→API Call
→Tool 執行（若有）→ 迴圈
→顯示回應 → 存入 SessionDB
📡 Gateway Message
01平台事件觸發
→Adapter.on_message()
→GatewayRunner 授權使用者
→解析 Session Key
→建立 AIAgent（含歷史）
→AIAgent.run_conversation()
→透過 Adapter 回傳結果
⏰ Cron Job
01Scheduler tick
→載入到期任務 jobs.json
→建立全新 AIAgent（無歷史）
→注入附加技能為 context
→執行任務 Prompt
→輸出到目標平台
→更新 next_run 狀態
