安全標頭總覽 Security Headers

add_header X-Frame-Options          "SAMEORIGIN" always;
add_header X-Content-Type-Options    "nosniff" always;
add_header Referrer-Policy           "strict-origin-when-cross-origin" always;
add_header Permissions-Policy        "camera=(), microphone=(), geolocation=()" always;
add_header Strict-Transport-Security "max-age=63072000; includeSubDomains; preload" always;
add_header Content-Security-Policy   "default-src 'self'; script-src 'self' 'unsafe-inline'" always;

# 隱藏版本號 & 後端標頭
server_tokens off;
proxy_hide_header X-Powered-By;
proxy_hide_header Server;

防盜連 ngx_http_referer_module

location ~* \.(jpg|jpeg|png|gif|webp|mp4)$ {
    valid_referers none blocked server_names *.example.com;
    if ($invalid_referer) { return 403; }
}
參數說明
none允許無 Referer（直接存取）
blockedReferer 被移除/隱藏
server_names匹配當前 server_name

SSL 硬化清單 SSL Hardening

項目建議
協定ssl_protocols TLSv1.2 TLSv1.3;
加密套件僅 AEAD（禁 CBC/RC4/3DES）
DH 參數ssl_dhparam 2048+ bit
HSTSmax-age ≥ 1 年
OCSPssl_stapling on;
Ticketsssl_session_tickets off;（完美前向保密）
曲線ssl_ecdh_curve X25519:secp256r1:secp384r1;

安全連結 ngx_http_secure_link_module

產生帶 MD5 簽章與過期時間的下載連結。

location /downloads/ {
    secure_link     $arg_md5,$arg_expires;
    secure_link_md5 "secret$uri$arg_expires";
    if ($secure_link = "")  { return 403; }
    if ($secure_link = "0") { return 410; }
}

Worker 調校 Core Module Tuning

worker_processes     auto;           # = CPU 核心數
worker_cpu_affinity  auto;           # 綁定 CPU
worker_rlimit_nofile 65535;          # 最大檔案描述符

events {
    worker_connections  16384;
    multi_accept        on;
    use                 epoll;
}
# 最大並發 = worker_processes × worker_connections
⚠️ 需配合 OS：ulimit -n 65535 與 /etc/security/limits.conf

stub_status 監控 ngx_http_stub_status_module

location /nginx_status {
    stub_status;
    allow 127.0.0.1; deny all;
}
指標說明
Active connections活躍連線數
accepts / handled已接受 / 已處理連線（相等=無丟棄）
requests已處理請求總數
Reading / Writing / Waiting讀標頭 / 送回應 / keepalive 閒置

實用小模組 Misc Modules

sub_filter（內容替換）
sub_filter 'http://' 'https://';
sub_filter_once off;
image_filter（即時縮圖）
image_filter resize 200 200;
image_filter_jpeg_quality 80;
realip（真實 IP）
set_real_ip_from 10.0.0.0/8;
real_ip_header   X-Forwarded-For;
real_ip_recursive on;
split_clients（A/B 測試）
split_clients $remote_addr $v {
    50%  .a;
    *    .b;
}
root /var/www/site$v;
