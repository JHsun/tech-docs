📚 API 文件 
### `getUser(id)`

取得指定 ID 的使用者。

#### Parameters
| 名稱 | 型別     | 必填 | 說明     |
|------|----------|------|----------|
| id   | `string` | ✅   | 使用者 ID|

#### Returns
```ts
Promise<User>
```

#### Example
```js
const user = await getUser("u_123");
```

#### Errors
- `404` 找不到使用者
- `401` 未授權
