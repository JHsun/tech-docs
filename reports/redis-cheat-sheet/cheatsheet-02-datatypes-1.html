S

String（字串）

最基礎型別
二進位安全 最大 512 MB 可存文字/數字/序列化物件
命令說明O()
SET key val [EX s] [NX|XX]設值。EX 過期秒數，NX 不存在才設，XX 存在才設O(1)
GET key取值O(1)
MSET k1 v1 k2 v2 ...批次設值O(N)
MGET k1 k2 ...批次取值O(N)
INCR key值 +1（原子計數器）O(1)
INCRBY key n值 +nO(1)
INCRBYFLOAT key f值 +f（浮點數）O(1)
DECR key / DECRBY值 -1 / -nO(1)
APPEND key val附加到現有值尾端O(1)
STRLEN key取得值長度O(1)
GETRANGE key start end取子字串O(N)
SETRANGE key offset val覆寫指定位置O(1)
SETNX key val不存在才設定（分散式鎖常用）O(1)
GETSET key val設新值並回傳舊值O(1)
GETDEL key取值後刪除O(1)
 
# 實用範例
SET user:1001:name "Alice" EX 3600
SET lock:order:42 token123 NX EX 30  # 分散式鎖
INCR page:views:home               # 計數器
MSET k1 "v1" k2 "v2" k3 "v3"    # 批次寫入
H

Hash（雜湊）

物件存儲
field-value 對 扁平結構（無巢狀） 最多 2³²-1 個 field
命令說明O()
HSET key f1 v1 [f2 v2 ...]設定一或多個 fieldO(N)
HGET key field取得單一 field 值O(1)
HMGET key f1 f2 ...批次取得多個 fieldO(N)
HGETALL key取得所有 field 與值O(N)
HDEL key f1 [f2 ...]刪除 fieldO(N)
HEXISTS key field檢查 field 是否存在O(1)
HLEN keyfield 數量O(1)
HKEYS key所有 field 名稱O(N)
HVALS key所有 field 值O(N)
HINCRBY key field nfield 值 +nO(1)
HINCRBYFLOAT key field ffield 值 +fO(1)
HSETNX key field valfield 不存在才設定O(1)
HSCAN key cursor ...漸進式迭代 fieldO(1)/次
HRANDFIELD key [count]隨機取 fieldO(N)
 
# 適合存儲物件
HSET user:1001 name "Alice" age 28 email "a@b.com"
HGET user:1001 name          # → "Alice"
HINCRBY user:1001 age 1    # → 29（生日！）
HGETALL user:1001           # → name Alice age 29 email a@b.com
✅
Hash 比多個 String key 更節省記憶體（小 Hash 使用 ziplist/listpack 編碼）。適合表示使用者、產品、設定等物件。
L

List（列表）

有序集合
雙向鏈結串列 依插入順序排列 允許重複 最多 2³²-1 元素
命令說明O()
LPUSH key v1 [v2 ...]從左側推入O(N)
RPUSH key v1 [v2 ...]從右側推入O(N)
LPOP key [count]從左側彈出O(N)
RPOP key [count]從右側彈出O(N)
LRANGE key start stop取得範圍元素（0 起始，-1 為末尾）O(S+N)
LINDEX key index依索引取值O(N)
LLEN key列表長度O(1)
LSET key index val依索引設值O(N)
LINSERT key BEFORE|AFTER pivot val在指定元素前/後插入O(N)
LREM key count val移除 count 個匹配元素O(N+M)
LTRIM key start stop只保留範圍內元素O(N)
LMOVE src dst L|R L|R原子地從 src 移動至 dstO(1)
BLPOP key [key ...] timeout阻塞式左彈出O(N)
BRPOP key [key ...] timeout阻塞式右彈出O(N)
 
# 佇列模式（FIFO）：RPUSH 入隊，LPOP 出隊
RPUSH queue:tasks "task1" "task2" "task3"
LPOP queue:tasks                # → "task1"

# 堆疊模式（LIFO）：RPUSH 推入，RPOP 彈出
RPUSH stack:undo "action1"
RPOP stack:undo                 # → "action1"

# 最新 N 筆記錄（capped list）
LPUSH log:latest "new entry"
LTRIM log:latest 0 99         # 僅保留前 100 筆

# 阻塞式消費（工作者等待任務）
BLPOP queue:tasks 30          # 等待最多 30 秒
S

Set（集合）

無序唯一
無序 元素唯一 支援集合運算 最多 2³²-1 元素
命令說明O()
SADD key m1 [m2 ...]新增成員O(N)
SREM key m1 [m2 ...]移除成員O(N)
SISMEMBER key member檢查是否為成員O(1)
SMISMEMBER key m1 m2 ...批次檢查多個成員O(N)
SMEMBERS key列出所有成員O(N)
SCARD key成員數量O(1)
SRANDMEMBER key [count]隨機取成員（不移除）O(N)
SPOP key [count]隨機彈出並移除O(N)
SMOVE src dst member移動成員至另一集合O(1)
集合運算
SUNION k1 k2 ...聯集O(N)
SINTER k1 k2 ...交集O(N×M)
SDIFF k1 k2 ...差集（k1 - k2）O(N)
S*STORE dst k1 k2運算結果存入 dstO(N)
SSCAN key cursor ...漸進式迭代O(1)/次
 
# 標籤系統
SADD article:1:tags "redis" "database" "nosql"
SADD article:2:tags "redis" "cache" "performance"

# 共同標籤（交集）
SINTER article:1:tags article:2:tags  # → "redis"

# 線上使用者追蹤
SADD online:users "user:1001"
SISMEMBER online:users "user:1001"  # → 1
SCARD online:users                   # → 在線人數
Z

Sorted Set（有序集合）

加權排序
唯一成員 + 浮點分數 依分數排序 底層：跳躍表 + 雜湊表 新增 O(log N)
命令說明O()
ZADD key score member [...]新增成員。支援 NX/XX/GT/LT/CHO(logN)
ZSCORE key member取得分數O(1)
ZRANK key member排名（低→高，0 起始）O(logN)
ZREVRANK key member排名（高→低）O(logN)
ZRANGE key min max [BYSCORE|BYLEX] [REV] [LIMIT]範圍查詢（Redis 6.2+ 統一語法）O(logN+M)
ZRANGEBYSCORE key min max依分數範圍查詢O(logN+M)
ZREVRANGE key start stop反向範圍（高→低）O(logN+M)
ZINCRBY key increment member增加分數O(logN)
命令說明O()
ZREM key member [...]移除成員O(M×logN)
ZCARD key成員總數O(1)
ZCOUNT key min max分數範圍內成員數O(logN)
ZPOPMIN key [count]彈出最低分成員O(logN×M)
ZPOPMAX key [count]彈出最高分成員O(logN×M)
BZPOPMIN / BZPOPMAX阻塞式彈出O(logN)
ZUNIONSTORE dst N k1 k2聯集並存儲O(N×logN)
ZINTERSTORE dst N k1 k2交集並存儲O(N×K×logN)
ZRANDMEMBER key [count]隨機取成員O(N)
ZSCAN key cursor ...漸進式迭代O(1)/次
 
# 排行榜
ZADD leaderboard 100 "Alice" 85 "Bob" 92 "Charlie"
ZINCRBY leaderboard 15 "Bob"            # Bob → 100

# Top 3（高→低）
ZREVRANGE leaderboard 0 2 WITHSCORES
# → Alice 100, Bob 100, Charlie 92

# 分數相同時按字典序，Alice 排 Bob 之前

# 速率限制器（滑動視窗）
ZADD ratelimit:user:1001 1711600000 "req-uuid-1"
ZRANGEBYSCORE ratelimit:user:1001 1711599940 +inf  # 最近60秒的請求
ZREMRANGEBYSCORE ratelimit:user:1001 -inf 1711599940  # 清理過期
💡
ZADD 條件選項：NX 僅新增不更新，XX 僅更新不新增，GT 新分數>舊分數才更新，LT 新分數<舊分數才更新，CH 回傳改變的元素數（含更新）。

五大基本型別速查比較

型別結構有序唯一底層編碼典型應用
String 單一值（text/int/binary） —— int, embstr, raw 快取、計數器、分散式鎖
Hash field → value 對映 否field 唯一 listpack, hashtable 使用者檔案、物件存儲
List 字串有序序列 ✅ 插入序否 listpack, quicklist 佇列、堆疊、最新動態
Set 字串集合 否✅ listpack, hashtable 標籤、去重、集合運算
Sorted Set 成員 + 分數 ✅ 分數序✅ listpack, skiplist+ht 排行榜、速率限制、優先佇列
