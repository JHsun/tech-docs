Playwright Test
playwright.config.ts 完整範本
playwright.config.ts
import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  testDir: './tests',                    // 測試目錄
  fullyParallel: true,                   // 所有測試平行
  forbidOnly: !!process.env.CI,          // CI 禁止 .only
  retries: process.env.CI ? 2 : 0,       // CI 重試 2 次
  workers: process.env.CI ? 1 : undefined,// CI 單 worker
  reporter: 'html',                      // 報告器
  timeout: 30_000,                       // 測試逾時 30s
  globalTimeout: 0,                      // 全域逾時（0=無限）
  expect: {
    timeout: 5_000,                      // 斷言逾時 5s
    toHaveScreenshot: { maxDiffPixels: 50 },
  },
  use: {
    baseURL: 'http://localhost:3000',     // 基礎 URL
    trace: 'on-first-retry',             // 追蹤錄製
    screenshot: 'only-on-failure',       // 失敗時截圖
    video: 'retain-on-failure',          // 失敗時錄影
    actionTimeout: 0,                    // 操作逾時
    navigationTimeout: 0,               // 導航逾時
    locale: 'zh-TW',                     // 語系
    timezoneId: 'Asia/Taipei',           // 時區
  },
  projects: [
    { name: 'chromium', use: { ...devices['Desktop Chrome'] } },
    { name: 'firefox',  use: { ...devices['Desktop Firefox'] } },
    { name: 'webkit',   use: { ...devices['Desktop Safari'] } },
    { name: 'mobile',   use: { ...devices['iPhone 13'] } },
  ],
  webServer: {
    command: 'npm run start',
    url: 'http://localhost:3000',
    reuseExistingServer: !process.env.CI,
    timeout: 120_000,
  },
});
use 選項完整對照表
選項類型預設值說明
baseURLstring—page.goto('/') 的基礎 URL
viewport{width, height}{1280, 720}頁面視窗大小
localestring'en-US'瀏覽器語系
timezoneIdstring—時區，如 'Asia/Taipei'
geolocation{longitude, latitude}—地理位置
permissionsstring[]—如 ['geolocation','notifications']
colorScheme'light'|'dark'|'no-preference'—色彩方案
storageStatestring|object—預載入驗證狀態
tracestring'off'追蹤錄製模式
videostring'off''on'|'off'|'retain-on-failure'|'on-first-retry'
screenshotstring'off''on'|'off'|'only-on-failure'
actionTimeoutnumber0（無限）操作逾時（ms）
navigationTimeoutnumber0（無限）導航逾時（ms）
testIdAttributestring'data-testid'自訂測試 ID 屬性
httpCredentials{username, password}—HTTP 基礎驗證
proxy{server, bypass?, username?, password?}—代理伺服器
ignoreHTTPSErrorsbooleanfalse忽略 HTTPS 錯誤
offlinebooleanfalse模擬離線
javaScriptEnabledbooleantrue啟用/停用 JavaScript
userAgentstring—自訂 User-Agent
extraHTTPHeadersRecord—附加 HTTP 標頭
hasTouchbooleanfalse支援觸控事件
isMobilebooleanfalse移動裝置模式
deviceScaleFactornumber1裝置像素比
launchOptionsobject—瀏覽器啟動選項
channelstring—'chrome'|'msedge' 等
headlessbooleantrue無頭模式
📝優先順序：test.use() > project.use > global use。使用 test.use({ locale: 'ja-JP' }) 在檔案或 describe 層級覆寫。
標註（Annotations）& Fixtures
🏷 標註方法
方法行為
test.skip()不執行，標記為跳過
test.skip(cond, reason)條件式跳過
test.fail()預期失敗（執行並驗證失敗）
test.fixme()待修復（不執行）
test.slow()逾時 ×3
test.only()僅執行此測試
標籤（Tags）： 
test('登入', { tag: ['@smoke', '@fast'] },
  async ({ page }) => { });

// 執行特定標籤
// npx playwright test --grep @smoke
// 排除標籤
// npx playwright test --grep-invert @slow
🐍Python：使用 pytest markers — @pytest.mark.skip、@pytest.mark.only_browser("chromium")、-m "smoke"
🧩 Fixtures 系統
內建 Fixtures（Node.js）：
Fixture範圍說明
pagetest隔離頁面
contexttest隔離上下文
browserworker共用瀏覽器
browserNameworker'chromium'|'firefox'|'webkit'
requesttestAPI 請求上下文
自訂 Fixture： 
export const test = base.extend<{
  todoPage: TodoPage
}>({
  todoPage: async ({ page }, use) => {
    const todo = new TodoPage(page);
    await todo.goto();   // Setup
    await use(todo);     // 提供給測試
    await todo.cleanup();// Teardown
  },
});
💡{ scope: 'worker' } Worker 範圍、{ auto: true } 自動、{ option: true } 可設定。
平行處理 · 專案 · 逾時 · 重試 · 報告
⚡ 平行處理 & 分片
// 全部平行
fullyParallel: true,

// 描述區塊內設定
test.describe.configure({
  mode: 'parallel' // parallel|serial|default
});

// Workers 數量
workers: 4,            // 固定數量
workers: '50%',        // CPU 百分比

// 分片（CI 用）
// npx playwright test --shard=1/4
📊testInfo.parallelIndex（0~workers-1，穩定）用於分配資源（如資料庫帳號）。
🐍Python 平行：pip install pytest-xdist && pytest --numprocesses auto

⏱ 逾時設定總覽
逾時類型預設設定方式
測試逾時30stimeout: 60_000（config）
test.setTimeout(60_000)（測試內）
Expect 逾時5sexpect: { timeout: 10_000 }
操作逾時∞use: { actionTimeout: 10_000 }
導航逾時∞use: { navigationTimeout: 30_000 }
全域逾時∞globalTimeout: 3_600_000
⏩test.slow() 將測試逾時乘以 3 倍。
🔄 重試 & 報告器
重試設定： 
retries: 2,              // 設定檔
// npx playwright test --retries=3  // CLI
// 測試內偵測重試次數
test('my test', async ({}, testInfo) => {
  if (testInfo.retry) { /* 重試邏輯 */ }
});
報告器類型：
報告器說明
list每行一測試（本機預設）
line單行進度更新
dot每測試一字元（CI 預設）
html自包含 HTML 報告
jsonJSON 輸出
junitJUnit XML
blob二進位（分片合併用）
reporter: [['html', { open: 'never' }], ['junit', { outputFile: 'results.xml' }]]
