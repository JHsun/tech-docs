第 1 章

什麼是 AWK？

AWK 是一種專為文字處理與資料擷取設計的程式語言，也是 Linux/Unix 系統中最經典的命令列工具之一。 它的名稱取自三位創造者的姓氏首字母：Alfred Aho、Peter Weinberger 與 Brian Kernighan，三人於 1977 年在 AT&T 貝爾實驗室開發了這個工具。

AWK 的核心設計理念是「模式-動作」（Pattern-Action）範式：逐行讀取輸入資料， 將每一行自動分割成欄位，再根據指定的模式決定是否執行對應的動作。 這讓它在處理結構化文字（如 CSV 檔案、日誌、設定檔、系統指令輸出等）時異常高效。

💡 AWK 與其他文字工具的比較
grep — 僅能搜尋與篩選符合模式的行
sed — 串流編輯器，擅長搜尋與取代
cut — 僅能擷取固定欄位
AWK — 涵蓋以上所有功能，還支援變數、函式、陣列、迴圈與條件判斷，是一個完整的程式語言

常見的 AWK 實作版本

版本說明預設系統
gawkGNU AWK，功能最豐富（支援網路、CSV、除錯器等）大部分 GNU/Linux 發行版
mawk注重執行速度，功能較精簡Debian / Ubuntu
nawkKernighan 的「新版 AWK」，即參考實作macOS / FreeBSD

在大多數系統上，輸入 awk 會自動連結到已安裝的實作版本。

第 2 章

基本語法與運作原理

呼叫格式

# 直接在命令列撰寫程式
awk [選項] '模式 { 動作 }' 輸入檔案

# 從檔案讀取程式
awk [選項] -f 程式檔案 輸入檔案

# 搭配管道使用
指令 | awk '模式 { 動作 }'

運作流程

讀取一筆記錄
（預設為一行）
→
分割成欄位
$1 $2 … $NF
→
測試所有模式
→
執行匹配的動作
→
下一筆記錄
重複直到結束

欄位變數

AWK 自動將每一行依分隔符號切割成欄位。$0 代表整行，$1 是第一個欄位， $2 是第二個，以此類推。$NF 永遠代表最後一個欄位。 

# 印出第 1 和第 3 個欄位
awk '{print $1, $3}' data.txt

# 印出包含 "error" 的行（省略動作 → 預設印出整行）
awk '/error/' logfile.txt

# 印出每行的最後一個欄位
awk '{print $NF}' file.txt

# 印出倒數第二個欄位
awk '{print $(NF-1)}' file.txt

# 搭配管道：找出 CPU 使用率超過 50% 的程序
ps aux | awk '$3 > 50 {print $11, $3"%"}'
✅ 重要規則
省略模式 → 對每一行都執行動作
省略動作 → 預設動作為 {print $0}（印出整行）
第 3 章

內建變數

記錄與欄位追蹤

變數說明範例
NR目前為止讀取的總記錄數（跨檔案累計）為每行編號：awk '{print NR, $0}'
NF目前記錄的欄位數量印出恰好 5 個欄位的行：awk 'NF==5'
FNR目前檔案內的記錄號（每個檔案重置）區分多檔案處理
FILENAME目前正在讀取的檔案名稱awk '{print FILENAME, $0}'
 
# 為每行加上行號
awk '{print NR, $0}' file.txt

# 只印出非空白行（NF 為 0 代表空行）
awk 'NF' file.txt

# 多檔案處理時區分檔案（FNR 會重置，NR 不會）
awk '{print FILENAME, FNR, NR, $0}' file1.txt file2.txt

分隔符號變數

變數說明預設值
FS輸入欄位分隔符（可為字元、字串或正則表達式）" "（連續空白）
RS輸入記錄分隔符"\n"（換行）
OFS輸出欄位分隔符（print 時插入各欄位之間）" "
ORS輸出記錄分隔符（每次 print 後附加）"\n"
 
# 解析以冒號分隔的 /etc/passwd
awk -F: '{print $1, $7}' /etc/passwd

# 設定輸出分隔符為逗號
awk -F: 'BEGIN {OFS=","} {print $1, $3, $6}' /etc/passwd

# 段落模式：以空行分隔記錄
awk 'BEGIN {RS=""} {print NR, $1}' document.txt

# 將換行替換成分號
awk 'BEGIN {ORS="; "} {print $0}' file.txt

其他重要變數

變數說明
RSTARTmatch() 函式匹配到的起始位置
RLENGTHmatch() 函式匹配到的長度
OFMT數字輸出格式（預設 "%.6g"）
SUBSEP多維陣列的下標分隔符（預設 "\034"）
ARGC / ARGV命令列引數數量與陣列

gawk 獨有的實用變數

# ENVIRON：存取環境變數
awk 'BEGIN {print ENVIRON["HOME"]}'

# FPAT：定義「欄位長什麼樣」而非「分隔符是什麼」
# 解決 CSV 中引號內含逗號的問題
gawk 'BEGIN {FPAT="([^,]*)|(\"[^\"]+\")"} {print $2}' data.csv

# IGNORECASE：設為非零值時，所有比對不分大小寫
gawk 'BEGIN {IGNORECASE=1} /error/' logfile.txt

第 5 章

模式匹配

BEGIN 與 END 區塊

BEGIN 在讀取任何輸入之前執行一次，適合初始化變數與印出表頭。 END 在所有輸入讀取完畢後執行一次，常用於輸出統計與摘要。 

awk 'BEGIN {FS=":"; print "使用者\t\tShell"}
     {print $1 "\t\t" $7}
     END {print "---", NR, "位使用者 ---"}' /etc/passwd

正則表達式模式

# 包含 "error" 的行
awk '/error/' logfile.txt

# 以數字開頭的行
awk '/^[0-9]/' data.txt

# 特定欄位匹配：第一個欄位包含 "admin"
awk '$1 ~ /admin/' users.txt

# 反向匹配：最後一個欄位不是 .log 檔案
awk '$NF !~ /\.log$/' filelist.txt

條件表達式模式

任何表達式都可以當作模式 — 結果為非零（數字）或非空（字串）時即匹配。

# 薪水超過 50000 的行
awk '$3 > 50000' employees.txt

# 印出偶數行
awk 'NR % 2 == 0' file.txt

# 印出長度超過 80 字元的行
awk 'length > 80' file.txt

範圍模式（Range Pattern）

# 印出 START 到 END 標記之間的所有內容（含首尾）
awk '/START/,/END/' file.txt

# 印出第 10 到第 20 行
awk 'NR==10, NR==20' file.txt

複合模式（邏輯運算子）

# 同時包含 "error" 和 "critical"
awk '/error/ && /critical/' logfile.txt

# 跳過註解和空行
awk '!/^#/ && NF' config.txt

# 複雜條件組合
awk 'NR > 1 && $3 > 1000 && $2 ~ /^A/' data.txt
第 6 章

動作與流程控制

print 與 printf

print 輸出後會自動以 OFS 分隔各引數、以 ORS（預設換行）結尾。 printf 則提供 C 語言風格的格式化輸出，不會自動換行。 

# print — 以逗號分隔引數（中間插入 OFS）
awk '{print $1, $3}' file.txt

# printf — 格式化表格輸出
awk '{printf "%-15s %10.2f\n", $1, $2}' data.txt

printf 常用格式說明符

說明符用途範例
%s字串printf "%s", $1
%d整數printf "%d", $2
%f浮點數printf "%.2f", $3
%e科學記號printf "%e", $4
%x十六進位printf "%x", 255 → ff
%-Ns靠左對齊，寬度 Nprintf "%-20s", $1
%0Nd前面補零printf "%05d", 42 → 00042

輸出重導向

print "data" > "output.txt"      # 寫入檔案（首次寫入時清空）
print "data" >> "output.txt"     # 附加到檔案
print "data" | "sort -u"         # 通過管道傳給指令

流程控制

# if-else 條件判斷
awk '{
    if ($3 > 90) grade = "A"
    else if ($3 > 80) grade = "B"
    else if ($3 > 70) grade = "C"
    else grade = "F"
    print $1, grade
}' scores.txt

# 三元運算子
awk '{print $1, ($3 > 60 ? "及格" : "不及格")}' scores.txt

# for 迴圈
awk 'BEGIN {for (i = 1; i <= 10; i++) print i}'

# while 迴圈
awk 'BEGIN {i=1; while (i<=5) {print i; i++}}'

# next：跳到下一筆記錄（常用於跳過標頭行）
awk 'NR == 1 {next} {print}' data.txt

# exit：找到第一筆匹配後立即結束
awk '/CRITICAL/ {print; exit}' logfile.txt
⚠️ 字串串接
AWK 的字串串接是隱式的 — 直接相鄰即可：
a = "hello" " " "world" → 結果為 "hello world"
不需要任何串接運算子。

賦值運算子一覽

AWK 支援所有常見的賦值運算子：=、+=、-=、*=、/=、%=、^=，以及遞增 ++ 與遞減 --。

內建函式

字串函式

函式說明
length(s)傳回字串長度（或陣列元素數）
substr(s, start, len)擷取子字串（從 1 開始計算）
index(s, target)傳回 target 在 s 中的位置（找不到傳回 0）
split(s, arr, sep)將字串分割存入陣列，傳回元素數
sub(regex, repl, target)取代第一個匹配（傳回 0 或 1）
gsub(regex, repl, target)取代所有匹配（傳回取代次數）
match(s, regex)尋找正則匹配，設定 RSTART、RLENGTH
sprintf(fmt, ...)格式化字串並傳回（不輸出）
tolower(s) / toupper(s)大小寫轉換
gensub(regex, repl, how, target)gawk 擴充：支援反向參照的通用取代
 
# substr：擷取第 5~7 個字元
awk 'BEGIN {print substr("washington", 5, 3)}'   # "ing"

# split：拆解分隔字串
awk 'BEGIN {
    n = split("蘋果-香蕉-橘子", a, "-")
    for (i=1; i<=n; i++) print a[i]
}'

# gsub：取代所有數字，& 代表匹配到的文字
awk '{gsub(/[0-9]+/, "[&]"); print}'
# 輸入 "第42行有3個錯誤" → "第[42]行有[3]個錯誤"

# match：從文字中擷取數字
echo "Error at line 42" | awk '{
    match($0, /[0-9]+/)
    print substr($0, RSTART, RLENGTH)
}'   # 輸出: 42

# gensub：使用反向參照交換兩個欄位（gawk）
echo "hello world" | gawk '{print gensub(/(.+) (.+)/, "\\2 \\1", "g")}'
# 輸出: world hello

數學函式

函式說明
sin(x) / cos(x)三角函式（弧度制）
atan2(y, x)反正切（常用 atan2(0,-1) 求 π）
sqrt(x)平方根
int(x)截斷取整（趨向零）
log(x) / exp(x)自然對數 / 指數

# 產生 1~6 的隨機整數（模擬骰子）

# 平方根

# 求 π
awk 'BEGIN {print atan2(0, -1)}'  # 3.14159…

I/O 函式

# getline：從管道讀取命令輸出到變數
awk 'BEGIN {
    "date" | getline d
    close("date")
    print "今天是:", d
}'

# getline：從另一個檔案逐行讀取
awk 'BEGIN {
    while ((getline line < "data.txt") > 0)
        print line
    close("data.txt")
}'

# system()：執行系統命令，傳回結束狀態碼
awk 'BEGIN {system("ls -la")}'
📌 getline 的回傳值
1 = 成功讀取一行    0 = 已到檔案結尾    -1 = 發生錯誤
務必在迴圈中使用 close() 關閉已開啟的檔案或管道。
第 8 章

自訂函式

AWK 支援使用者定義函式，語法為 function 名稱(參數) { ... }。 由於 AWK 沒有 local 關鍵字，區域變數的慣用做法是將它們列為額外的參數， 並以多個空格與實際參數分隔 — 這就是著名的「空格技巧」。 

# 階乘（遞迴）
awk '
function factorial(n) {
    if (n <= 1) return 1
    return n * factorial(n - 1)
}
BEGIN {print factorial(10)}'   # 3628800

# 字串反轉（含區域變數技巧）
awk '
function reverse(s,    i, result) {
    result = ""
    for (i = length(s); i >= 1; i--)
        result = result substr(s, i, 1)
    return result
}
BEGIN {print reverse("hello")}'   # olleh

# 連接陣列元素
awk '
function join(arr, n, sep,    i, result) {
    result = arr[1]
    for (i = 2; i <= n; i++)
        result = result sep arr[i]
    return result
}
BEGIN {
    n = split("台北:台中:高雄", parts, ":")
    print join(parts, n, " → ")
}'   # 台北 → 台中 → 高雄
✅ 參數傳遞規則
純量（數字與字串）— 以值傳遞（call by value）
陣列 — 以參照傳遞（call by reference）
第 9 章

關聯式陣列

AWK 中所有陣列都是關聯式陣列（Associative Array），也就是以字串為鍵的雜湊表。 它們不需要事先宣告、可以動態增長，是 AWK 最強大的功能之一。 

# 計算詞頻
awk '{
    for (i=1; i<=NF; i++) freq[$i]++
}
END {
    for (w in freq) print freq[w], w
}' file.txt | sort -rn

# 群組彙總（類似 SQL 的 GROUP BY）
awk '{
    dept_total[$2] += $3
    dept_count[$2]++
}
END {
    for (d in dept_total)
        printf "%-12s 平均=%.0f\n", d, dept_total[d]/dept_count[d]
}' employees.txt

陣列操作

操作語法說明
新增 / 修改arr["key"] = value直接賦值即可
檢查存在if ("key" in arr)不會產生新元素
刪除元素delete arr["key"]移除單一元素
刪除整個陣列delete arr清空所有元素
走訪所有元素for (k in arr)順序不保證

模擬多維陣列

透過 SUBSEP（預設為 \034）串接下標來模擬多維陣列：

# 二維彙總
awk '{sales[$1, $2] += $3}
END {
    for (key in sales) {
        split(key, idx, SUBSEP)
        printf "%s %s: %.2f\n", idx[1], idx[2], sales[key]
    }
}' sales.txt

📌 NR==FNR 慣用模式解說

常用單行指令速查

以下是每位 AWK 使用者都應該熟記的經典單行指令：

# ★ 移除重複行（保留順序）— 最經典的 awk 單行指令
awk '!seen[$0]++' file.txt

# 移除空白行
awk 'NF' file.txt

# 為每行加上右對齊行號
awk '{printf "%5d : %s\n", NR, $0}' file.txt

# 印出每行的最後一個欄位
awk '{print $NF}' file.txt

# 反轉整個檔案的行順序（等同 tac）
awk '{a[NR]=$0} END {for(i=NR;i>=1;i--) print a[i]}' file.txt

# 計算行數（等同 wc -l）
awk 'END {print NR}' file.txt

# 去除行首與行尾的空白字元
awk '{gsub(/^[ \t]+|[ \t]+$/, ""); print}' file.txt

# 雙倍行距
awk '1; {print ""}' file.txt

# 印出匹配行的下一行
awk '/pattern/ {getline; print}' file.txt

# 印出匹配行的前一行
awk '/pattern/ {print prev} {prev=$0}' file.txt

# 詞頻統計
awk '{for(i=1;i<=NF;i++) freq[$i]++}
END {for(w in freq) print freq[w], w}' file.txt | sort -rn

# 只印出第 N 到第 M 行
awk 'NR>=5 && NR<=10' file.txt

# 擷取所有 IP 位址
awk '{
    while(match($0,/[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+/)) {
        print substr($0,RSTART,RLENGTH)
        $0=substr($0,RSTART+RLENGTH)
    }
}' log.txt

# 計算檔案的欄位總和（假設第 2 欄）
awk '{sum += $2} END {print sum}' data.txt
✅ 理解 !seen[$0]++
這是 AWK 中最精妙的慣用寫法。運作原理：
1. seen[$0] 首次遇到某行時值為 0（假）
2. ! 取反後為真 → 印出該行
3. ++ 隨後將值加 1
4. 再次遇到相同行時 seen[$0] 已非零 → 取反後為假 → 不印出
巧妙地在一個表達式中同時完成「首次出現時印出」和「標記為已見過」。
