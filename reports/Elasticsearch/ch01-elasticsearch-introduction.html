CH 01 · Elasticsearch 概論與 Elastic Stack 全貌

適合對象 🟢 初學者
閱讀時間 約 15-20 分鐘
本章目標 理解 Elasticsearch 的本質、ELK/Elastic Stack 的演進、Self-managed 與 Cloud 部署的差異，以及最新的授權模式

什麼是 Elasticsearch？

Elasticsearch 是一套分散式（distributed）、近即時（near real-time）的搜尋與分析引擎，由 Elastic 公司主導開發。它建立在 Apache Lucene 之上，但提供了 Lucene 所沒有的關鍵能力：

  • 分散式架構：可橫向擴展（horizontally scalable）至數百個節點，支援 PB 級資料
  • RESTful API：所有操作（索引、搜尋、管理）均透過 HTTP/JSON 進行
  • Schema-flexible：支援動態 mapping，但也允許嚴格 schema 控制
  • 多用途資料儲存：可同時當作 search engine、analytics engine、time-series database、vector database 使用

從 2010 年首次釋出到今日（2026 年），Elasticsearch 已成為全球前 10 大最受歡迎的資料庫之一，被超過 50% 的 Fortune 500 企業使用。

💡 一句話定義 Elasticsearch = Lucene（搜尋核心）+ 分散式叢集（distributed cluster）+ RESTful API + JSON 文件儲存。

Elasticsearch 能做什麼？

Elasticsearch 並非單一用途的資料庫。它是一個多模引擎，可同時應用於以下情境：

使用情境說明典型客戶
全文檢索（Full-text Search）網站搜尋、電商商品搜尋、企業文件搜尋Wikipedia、Shopify、Adobe
日誌與可觀測性（Logs / Observability）集中收集應用程式 / 系統 / 容器日誌、APM、Traces、MetricsUber、Netflix、CERN
資安分析（SIEM / Security Analytics）威脅偵測、入侵分析、合規稽核美國國防部、各大金融業
商業分析（Business Analytics）即時 BI 儀表板、KPI 監控Walmart、eBay
地理空間查詢（Geospatial）距離搜尋、區域分析、IoT 軌跡Uber Eats、Foursquare
向量搜尋與 GenAI（Vector / RAG）語義搜尋、相似度比對、RAG 應用後端Roboflow、各大 LLM 應用

Elastic Stack（ELK）的演進

許多人聽過「ELK」這個名詞，它代表早期 Elastic 的三大核心：

  • Elasticsearch — 搜尋與儲存核心
  • Logstash — 資料收集與管線處理
  • Kibana — 視覺化與管理 UI

後來加入 Beats（輕量級 data shipper），ELK 演變成「Elastic Stack」。近年又加入 Elastic Agent 與 OpenTelemetry Collector，整體生態系如下：

部署選擇：Self-managed vs Cloud

Elastic 提供 5 種部署方式，本教學文件聚焦在 Self-managed：

部署模式說明適用情境
Self-managed ⭐自己安裝、自己管理（本教學主題）地端部署、完全掌控、特殊合規要求、成本敏感
Elastic Cloud Hosted (ECH)Elastic 在 AWS/GCP/Azure 上代管快速上線、不想維運基礎設施
Elastic Cloud Serverless完全 Serverless，按用量計費不可預測流量、新創、PoC
Elastic Cloud Enterprise (ECE)地端 orchestrator，可管多個 deployments大型企業 / MSP、需要私有雲多租戶
Elastic Cloud on Kubernetes (ECK)K8s Operator，CRD 管理 ES/Kibana已有 K8s 平台、雲原生工作負載

為什麼選擇 Self-managed？

  • 完全掌控：硬體、網路、版本、外掛、調校全自己決定
  • 地端 / Air-gapped：金融、政府、軍工等需與外網隔離的環境
  • 成本最低：對大規模長期運行而言，授權 + 自有硬體成本通常低於 Cloud 方案
  • 客製化：可安裝自訂 plugin、修改原始碼（在授權允許範圍內）
  • 整合既有基礎設施：與企業內 LDAP、AD、SAML、SIEM、CMDB 等深度整合

授權模式（Licensing）

Elasticsearch 的授權歷史較複雜，以下是 2026 年最新狀態的整理：

原始碼授權（Source code license）

從 2024 年 9 月起，Elasticsearch 與 Kibana 的原始碼提供三選一授權（triple-license），使用者可挑選最適合自己情境的一種：

授權OSI 認可主要限制適合情境
AGPL v3✅ 是修改後公開原始碼（含 SaaS 場景）看重開放原始碼純粹性
SSPL 1.0❌ 否提供 ES 為 SaaS 必須開源整個服務管線類似 MongoDB 用戶
ELv2❌ 否（source-available）不可作為托管服務轉售、不可繞過授權檢查大多數企業內部使用

二進位發行版授權（Default distribution）

Elastic 官網下載的 binary（.tar.gz、.deb、.rpm、Docker image）一律採用 Elastic License v2 (ELv2)。它是 source-available 授權，提供：

  • 免費使用所有 Basic（基礎）功能，包括 Security 基本功能、ILM、Snapshot 等
  • 不可將 Elasticsearch 包裝成托管服務（managed service）轉售給第三方
  • 不可繞過授權驗證機制（例如改 code 解鎖付費功能）
  • 不可移除或變更 Elastic 商標、版權聲明
✅ 對 99% 使用者完全免費 如果你只是在公司內部或專案中使用 Elasticsearch（即使是用於商業軟體後端），ELv2 完全允許。只要不把它當成 SaaS 賣給別人，就沒有任何成本。

付費訂閱（Subscription tiers）

Elastic 將功能分為 4 個訂閱等級：

等級價格包含功能
Basic免費（ELv2）核心 ES、Kibana、Security 基礎、ILM、Snapshot、ES|QL、向量搜尋、ELSER、Canvas、Maps、單叢集 Stack Monitoring
Gold付費+ JDBC/ODBC client、官方支援
Platinum付費+ Machine Learning、Graph、Cross-cluster Replication、Audit Logging、進階安全（SAML、Kerberos、欄位/文件層級安全）、Watcher、進階 connectors
Enterprise付費+ Searchable Snapshots、Frozen tier、AutoOps、Endpoint Protection、SOAR、進階 ML/AI 功能

完整功能對照表可參考：Elastic Subscriptions

當前版本與發布節奏

Elastic Stack 採語意化版本（Semantic Versioning），當前最新狀態（2026 年 4 月）：

  • 9.3.4 — 最新 GA 版本（包含關鍵修復）
  • 9.0 LTS — 9.x 系列首版，2026 年 4 月發布
  • 8.19.x — 8.x 系列維護版本
  • 7.17.x — 7.x 系列僅安全更新（已停止主要開發）

版本支援政策（EOL）

Elastic 對每個 major 版本提供以下支援期限（取兩者中較長者）：

  • 該 major GA 後 30 個月，或
  • 下一個 major GA 後 18 個月

例如 9.0 於 2026/04 GA，那麼 9.x 的支援至少持續至 2028/10。建議生產環境永遠使用最新的 minor + 最新 patch，並每 1-2 年規劃一次 major 升級。

Elastic Stack 元件版本一致性

🚨 鐵則：所有元件版本必須一致 Elasticsearch、Kibana、Logstash、Beats、Elastic Agent、APM Server 必須使用完全相同的版本號。混用版本可能導致 API 不相容、功能異常、甚至無法啟動。

本章重點回顧

  • Elasticsearch 是分散式、近即時的搜尋與分析引擎，建立在 Lucene 之上
  • 它不只是「搜尋引擎」，也是 analytics、time-series、vector database
  • ELK Stack 已演進為 Elastic Stack：核心 = ES + Kibana；資料蒐集已轉向 Elastic Agent + EDOT；Logstash 仍是進階 ETL 首選
  • 共有 5 種部署模式，本教學專注 Self-managed
  • 2024/9 起原始碼新增 AGPL v3 授權選項，二進位發行版仍為 ELv2，對大多數使用者完全免費
  • 當前最新版本：Elastic Stack 9.3.4（2026/04）
  • 所有 Stack 元件版本必須一致
