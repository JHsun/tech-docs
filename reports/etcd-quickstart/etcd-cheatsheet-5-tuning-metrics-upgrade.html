⚡效能調校 — 時間參數

參數預設建議
--heartbeat-interval100ms0.5–1.5× RTT
--election-timeout1000ms≥10× RTT，上限 50s
etcd --heartbeat-interval=100 --election-timeout=500
過短 → 頻繁選舉；過長 → 偵測慢。

💾磁碟 & 網路調校

磁碟 I/O

sudo ionice -c2 -n0 -p $(pgrep etcd)

WAL+snap 分兩 SSD → 提升 ~20%。

網路優先 (tc)

tc qdisc add dev eth0 root handle 1: prio bands 3
tc filter add dev eth0 parent 1:0 protocol ip prio 1 \
  u32 match ip sport 2380 0xffff flowid 1:1
tc filter add dev eth0 parent 1:0 protocol ip prio 1 \
  u32 match ip dport 2380 0xffff flowid 1:1
tc filter add dev eth0 parent 1:0 protocol ip prio 2 \
  u32 match ip sport 2379 0xffff flowid 1:1
tc filter add dev eth0 parent 1:0 protocol ip prio 2 \
  u32 match ip dport 2379 0xffff flowid 1:1
