🌊 Stream JSON Lines (≥ 0.134.0)

自動以 application/jsonl 格式串流回應。

from collections.abc import AsyncIterable

@app.get("/items/stream")
async def stream_items() -> AsyncIterable[Item]:
    for item in items:
        yield item  # 每個 item 一行 JSON

StreamingResponse（通用串流）

from fastapi.responses import StreamingResponse

async def generate():
    for i in range(100):
        yield f"data: {i}\n"
        await asyncio.sleep(0.1)

@app.get("/stream")
async def stream():
    return StreamingResponse(
        generate(), media_type="text/plain"
    )

📡 Server-Sent Events (≥ 0.135.0)

from fastapi.sse import EventSourceResponse, ServerSentEvent

@app.get("/sse", response_class=EventSourceResponse)
async def sse_stream() -> AsyncIterable[ServerSentEvent]:
    yield ServerSentEvent(comment="starting")
    for i, item in enumerate(items):
        yield ServerSentEvent(
            data=item,           # 自動 JSON 編碼
            event="update",
            id=str(i),
            retry=5000,          # 重連延遲(ms)
        )

ServerSentEvent 參數

參數說明
data自動 JSON 編碼的資料
raw_data原始字串（與 data 互斥）
event事件名稱
id事件 ID
retry重連延遲（毫秒）
commentSSE 註解
ℹ️
data 和 raw_data 互斥，不能同時使用。預設每 15 秒自動發送 keep-alive ping。

⏳ 背景任務 — Background Tasks

from fastapi import BackgroundTasks

def write_log(message: str):
    with open("log.txt", "a") as f:
        f.write(message)

@app.post("/send/{email}")
async def send(
    email: str,
    background_tasks: BackgroundTasks
):
    background_tasks.add_task(
        write_log,
        f"Notification for {email}\n"
    )
    return {"message": "Sent"}
⚡
背景任務在所有中介層之後執行。
繁重運算（ML 推論、大量資料處理）應使用 Celery，背景任務適合輕量工作（寄信、寫 log）。

🧪 測試 — Testing

基本測試（同步）

from fastapi.testclient import TestClient

client = TestClient(app)

def test_read_main():
    response = client.get(
        "/items/foo",
        headers={"X-Token": "secret"}
    )
    assert response.status_code == 200
    assert response.json() == {"id": "foo"}

def test_create_item():
    response = client.post(
        "/items/",
        json={"id": "new", "title": "New"},
        headers={"X-Token": "secret"}
    )
    assert response.status_code == 200

非同步測試（httpx + anyio）

import pytest
from httpx import ASGITransport, AsyncClient

@pytest.mark.anyio
async def test_root():
    async with AsyncClient(
        transport=ASGITransport(app=app),
        base_url="http://test"
    ) as ac:
        response = await ac.get("/")
    assert response.status_code == 200

特殊測試場景

場景方法
WebSocketwith client.websocket_connect("/ws") as ws
Lifespan 事件with TestClient(app) as client:（觸發 startup/shutdown）
依賴覆蓋app.dependency_overrides[original] = mock_fn
測試後清理app.dependency_overrides = {}
⚠️
TestClient 不能用在 async def 測試函式中。
AsyncClient 不會觸發 lifespan 事件 — 需額外使用 asgi-lifespan 套件。

@app.get("/example/")

OpenAPI 額外回應定義

@app.get("/items/{id}",

💡

ORJSONResponse 和 UJSONResponse 已不再建議使用。

⚙️ 設定與環境變數

from functools import lru_cache
from pydantic_settings import BaseSettings, SettingsConfigDict

class Settings(BaseSettings):
    app_name: str = "My API"
    admin_email: str
    items_per_user: int = 50
    model_config = SettingsConfigDict(
        env_file=".env"
    )

@lru_cache
def get_settings():
    return Settings()

@app.get("/info")
async def info(
    settings: Annotated[Settings, Depends(get_settings)]
):
    return {"app_name": settings.app_name}
⚠️
@lru_cache 必不可少！沒有它，每次請求都會重新讀取 .env 檔案。
環境變數優先於 .env 檔案中的值。
測試時覆蓋：app.dependency_overrides[get_settings] = lambda: Settings(admin_email="test@test.com")

🐳 部署 — Deployment

執行指令

場景指令
開發fastapi dev main.py
生產fastapi run main.py
多 Workerfastapi run --workers 4 main.py
Uvicorn 直接uvicorn main:app --host 0.0.0.0 --port 80 --workers 4
⚠️
生產環境絕對不要使用 --reload。

Dockerfile（最佳實踐）

FROM python:3.14

WORKDIR /code

# 先複製 requirements → 安裝 → 再複製程式碼
# 利用 Docker Layer Cache 優化建置速度
COPY ./requirements.txt /code/requirements.txt
RUN pip install --no-cache-dir --upgrade \
    -r /code/requirements.txt

COPY ./app /code/app

# 必須使用 exec 格式（JSON 陣列）！
CMD ["fastapi", "run", "app/main.py", "--port", "80"]

# 反向代理後面加：--proxy-headers
# 多 Worker：--workers 4

六大部署概念

#概念建議方案
1HTTPSTLS 終止代理（Traefik/Caddy 自動續約 Let's Encrypt）
2開機啟動Docker / Kubernetes / systemd
3自動重啟Docker --restart / K8s Pod 重啟策略
4多副本--workers N 或 K8s replicas（推薦每容器 1 程序）
5記憶體N workers × 每程序記憶體 = 總 RAM 需求
6前置步驟DB migration 透過 init container 或腳本執行一次
⚠️
Docker CMD 必須用 exec 格式 ["fastapi", "run", ...]！Shell 格式 CMD fastapi run ... 會阻止容器正常關閉（signal 無法傳遞）。

Kubernetes：每個容器 1 個程序，讓叢集管理副本。已棄用的 tiangolo/uvicorn-gunicorn-fastapi 映像不應再使用。

💀 十大關鍵陷阱總整理

#陷阱正確做法
1Field 從 fastapi 匯入from pydantic import Field
2str | None 以為是非必填必須加 = None 預設值才是非必填
3Depends(func()) 呼叫了函式Depends(func) 傳入可呼叫物件
4Form/File 混用 JSON BodyHTTP 協議限制，同一端點只能選一種
5allow_credentials=True + ["*"]CORS credentials 啟用時必須明確列出來源
6PATCH 不用 exclude_unset=True預設值會覆蓋已存在的資料
7yield 依賴 except 不重新 raise必須 re-raise，否則客戶端得到無聲 500
8密碼比較用 ==用 secrets.compare_digest() 防計時攻擊
9Docker CMD 用 shell 格式必須用 exec 格式 ["fastapi", "run", ...]
10Settings 不加 @lru_cache每次請求都重新讀取 .env 檔案
