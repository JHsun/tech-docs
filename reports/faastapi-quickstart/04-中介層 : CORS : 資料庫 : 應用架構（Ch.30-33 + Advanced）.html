⛓️ 中介層 — Middleware

在每個請求/回應之前/之後執行的程式碼。

import time
from fastapi import Request

@app.middleware("http")
async def add_process_time(
    request: Request,
    call_next
):
    start = time.perf_counter()
    response = await call_next(request)
    process_time = time.perf_counter() - start
    response.headers["X-Process-Time"] = str(process_time)
    return response
ℹ️
執行順序：最後添加的中介層是最外層（請求先經過，回應最後經過）。

🌐 CORS 跨域資源共享

from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
    expose_headers=["X-Custom"],
    max_age=600,
)

參數說明

參數說明
allow_origins允許的來源列表
allow_credentials允許攜帶 Cookie
allow_methods允許的 HTTP 方法
allow_headers允許的請求標頭
expose_headers瀏覽器可見的回應標頭
max_age預檢快取秒數
⚠️
致命陷阱：若 allow_credentials=True，則 allow_origins、allow_methods、allow_headers 不能使用 ["*"]！必須明確列出。瀏覽器會直接拒絕。

🗄️ SQL 資料庫 — SQLModel 整合

官方推薦使用 SQLModel（結合 SQLAlchemy + Pydantic）。

from sqlmodel import Field, Session, SQLModel, create_engine, select

# ── 模型定義（多層繼承）──
class HeroBase(SQLModel):
    name: str = Field(index=True)
    age: int | None = Field(default=None, index=True)

class Hero(HeroBase, table=True):     # 資料庫表
    id: int | None = Field(default=None, primary_key=True)
    secret_name: str

class HeroPublic(HeroBase):           # 回應模型（不含 secret_name）
    id: int

class HeroCreate(HeroBase):           # 建立用輸入模型
    secret_name: str

class HeroUpdate(HeroBase):           # 部分更新模型
    name: str | None = None
    age: int | None = None
    secret_name: str | None = None

# ── Engine 與 Session ──
engine = create_engine(
    "sqlite:///database.db",
    connect_args={"check_same_thread": False}  # SQLite 必需！
)

def get_session():
    with Session(engine) as session:
        yield session

SessionDep = Annotated[Session, Depends(get_session)]

# ── CRUD 路由 ──
@app.post("/heroes/", response_model=HeroPublic)
def create_hero(hero: HeroCreate, session: SessionDep):
    db_hero = Hero.model_validate(hero)
    session.add(db_hero)
    session.commit()
    session.refresh(db_hero)
    return db_hero

@app.get("/heroes/", response_model=list[HeroPublic])
def read_heroes(session: SessionDep, offset: int = 0, limit: Annotated[int, Query(le=100)] = 100):
    return session.exec(select(Hero).offset(offset).limit(limit)).all()

@app.patch("/heroes/{hero_id}", response_model=HeroPublic)
def update_hero(hero_id: int, hero: HeroUpdate, session: SessionDep):
    db_hero = session.get(Hero, hero_id)
    if not db_hero: raise HTTPException(status_code=404)
    hero_data = hero.model_dump(exclude_unset=True)
    db_hero.sqlmodel_update(hero_data)
    session.add(db_hero); session.commit(); session.refresh(db_hero)
    return db_hero
⚡
check_same_thread=False 是 SQLite 專用，因為 FastAPI 在不同執行緒處理請求。PostgreSQL/MySQL 不需要。

SQLModel 模型分層設計

模型用途關鍵差異
HeroBase共用欄位基礎類別不含 id、不含 table
Hero (table=True)資料庫表映射含 id (PK) + secret_name
HeroPublicAPI 回應含 id、不含 secret_name
HeroCreate建立輸入含 secret_name、不含 id
HeroUpdate部分更新所有欄位可選

📂 APIRouter — 大型應用架構

app/ ├── __init__.py ├── main.py ← app = FastAPI() ├── dependencies.py ← 共用依賴 └── routers/ ├── __init__.py ├── items.py ← APIRouter └── users.py ← APIRouter
# routers/items.py
from fastapi import APIRouter

router = APIRouter(
    prefix="/items",
    tags=["items"],
    dependencies=[Depends(get_token_header)],
    responses={404: {"description": "Not found"}},
)

@router.get("/")      # 實際路徑: /items/
async def read_items():
    return [{"name": "Foo"}]

# main.py
from .routers import items, users

app.include_router(items.router)
app.include_router(
    users.router,
    prefix="/admin",
    tags=["admin"],
    dependencies=[Depends(verify_admin)]
)
⚠️
prefix 不能以 / 結尾！prefix="/items/" 會導致雙斜線。
Tags 和 responses 在 include_router 時是累加的，不會覆蓋。

🔌 WebSocket

from fastapi import WebSocket, WebSocketDisconnect

@app.websocket("/ws/{client_id}")
async def websocket_endpoint(
    websocket: WebSocket,
    client_id: int,
):
    await websocket.accept()
    try:
        while True:
            data = await websocket.receive_text()
            await websocket.send_text(f"Echo: {data}")
    except WebSocketDisconnect:
        print(f"Client {client_id} left")

WebSocket 方法

方法說明
accept()接受連線
receive_text()接收文字
receive_bytes()接收二進位
receive_json()接收 JSON
send_text(data)發送文字
send_bytes(data)發送二進位
send_json(data)發送 JSON
close(code=1000)關閉連線
ℹ️
WebSocket 支援 Depends、Security、Cookie、Header、Path、Query。
錯誤用 WebSocketException（不是 HTTPException）。

🔄 Lifespan 生命週期事件

取代已棄用的 @app.on_event()。

from contextlib import asynccontextmanager

ml_models = {}

@asynccontextmanager
async def lifespan(app: FastAPI):
    # ── 啟動 ──
    ml_models["model"] = load_model()
    yield
    # ── 關閉 ──
    ml_models.clear()

app = FastAPI(lifespan=lifespan)
⚡
不能混用 lifespan 與 @app.on_event()。
Lifespan 事件只在主應用上執行，不會傳遞到 mount 的子應用。

📄 Templates 與靜態檔案

Jinja2 模板

from fastapi.templating import Jinja2Templates
from fastapi.responses import HTMLResponse

templates = Jinja2Templates(directory="templates")

@app.get("/items/{id}", response_class=HTMLResponse)
async def read_item(request: Request, id: str):
    return templates.TemplateResponse(
        request=request,  # 必須傳入！
        name="item.html",
        context={"id": id}
    )

靜態檔案

from fastapi.staticfiles import StaticFiles

app.mount(
    "/static",
    StaticFiles(directory="static"),
    name="static"
)

內建 ASGI 中介層

中介層功能
HTTPSRedirectMiddlewareHTTP → HTTPS 重導向
TrustedHostMiddleware限制允許的 Host 標頭
GZipMiddleware壓縮回應（min_size / compresslevel）
from fastapi.middleware.gzip import GZipMiddleware
from fastapi.middleware.trustedhost import TrustedHostMiddleware

app.add_middleware(GZipMiddleware, minimum_size=1000)
app.add_middleware(TrustedHostMiddleware,
    allowed_hosts=["example.com", "*.example.com"])
💡
永遠用 app.add_middleware() 添加，不要手動包裝 ASGI app。
