🧱 依賴注入核心 — Dependency Injection

FastAPI 內建的 DI 系統是其最強大的功能之一。依賴可以是函式、類別或任何可呼叫物件。

from typing import Annotated
from fastapi import Depends

async def common_parameters(
    q: str | None = None,
    skip: int = 0,
    limit: int = 100,
):
    return {"q": q, "skip": skip, "limit": limit}

# 建立可重用的型別別名
CommonsDep = Annotated[dict, Depends(common_parameters)]

@app.get("/items/")
async def read_items(commons: CommonsDep):
    return commons

@app.get("/users/")
async def read_users(commons: CommonsDep):
    return commons
⚠️
最常犯的錯誤：Depends(func) 傳入可呼叫物件，不是 Depends(func()) 呼叫結果！
✅ Depends(common_parameters)
❌ Depends(common_parameters())

依賴注入層級架構

FastAPI(dependencies=[...]) → 全部路由（全域）
↓
APIRouter(dependencies=[...]) → 該 Router 所有路由
↓
@app.get("/", dependencies=[...]) → 單一路由（回傳值丟棄）
↓
param: Annotated[T, Depends(dep)] → 單一路由（回傳值注入）

🏗️ 類別依賴

類別的 __init__ 參數即為依賴的參數。

class CommonQueryParams:
    def __init__(
        self,
        q: str | None = None,
        skip: int = 0,
        limit: int = 100,
    ):
        self.q = q
        self.skip = skip
        self.limit = limit

@app.get("/items/")
async def read_items(
    commons: Annotated[CommonQueryParams, Depends()]
    # 簡寫！型別就是依賴，Depends() 自動推斷
):
    return {"q": commons.q}
💡
快捷寫法：Annotated[ClassName, Depends()] 等同於 Annotated[ClassName, Depends(ClassName)]，從型別自動推斷。

🔗 子依賴與快取

依賴可以有自己的依賴，形成依賴樹。

def query_extractor(q: str | None = None):
    return q

def query_or_cookie_extractor(
    q: Annotated[str, Depends(query_extractor)],
    last_query: Annotated[str | None, Cookie()] = None,
):
    if not q:
        return last_query
    return q

依賴快取行為

設定行為
use_cache=True（預設）同一請求內，相同依賴只呼叫一次
use_cache=False每次注入都重新呼叫
# 強制每次都重新呼叫
dep: Annotated[T, Depends(my_dep, use_cache=False)]

♻️ Yield 依賴 — 清理模式

用於資料庫連線、檔案操作等需要清理的場景。

async def get_db():
    db = DBSession()
    try:
        yield db         # ← 注入的值
    finally:
        db.close()      # ← 清理（回應送出後執行）
yield 之前的程式碼 → 啟動
↓
yield 的值 → 注入到路由函式
↓
yield 之後的程式碼 → 清理（回應已送出）
⚠️
重要：except 區塊中必須重新拋出例外（raise），否則客戶端會得到無聲的 500 錯誤。

scope="function"：清理會在回應送出前執行（用於必須在回應前釋放的資源）。

Depends() 完整參數

Depends(
    dependency=None,    # 可呼叫物件或 None（從型別推斷）
    use_cache=True,     # 同一請求快取
    scope="request",     # "request" 或 "function"
)

🎯 依賴作用域與裝飾器依賴

裝飾器依賴（回傳值不注入）

async def verify_token(x_token: Annotated[str, Header()]):
    if x_token != "fake-secret":
        raise HTTPException(status_code=400, detail="Invalid X-Token")

async def verify_key(x_key: Annotated[str, Header()]):
    if x_key != "fake-key":
        raise HTTPException(status_code=400, detail="Invalid X-Key")

# 裝飾器上的依賴：適合驗證/授權（不需要回傳值）
@app.get("/items/", dependencies=[Depends(verify_token), Depends(verify_key)])
async def read_items():
    return [{"item": "Foo"}]

全域依賴

app = FastAPI(dependencies=[Depends(verify_token)])
# 所有路由都會執行 verify_token

參數化可呼叫依賴（進階）

class FixedContentQueryChecker:
    def __init__(self, fixed_content: str):
        self.fixed_content = fixed_content

    def __call__(self, q: str = ""):
        return self.fixed_content in q if q else False

checker = FixedContentQueryChecker("bar")  # 預先配置

@app.get("/check/")
async def check(result: Annotated[bool, Depends(checker)]):
    return {"contains_bar": result}

🔑 Security — OAuth2 起步

FastAPI 安全性基於依賴注入系統。OAuth2PasswordBearer 是最常用的方案。

from fastapi.security import OAuth2PasswordBearer, OAuth2PasswordRequestForm

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="token")

@app.get("/users/me")
async def read_users_me(
    token: Annotated[str, Depends(oauth2_scheme)]
):
    return {"token": token}

依賴鏈設計模式

OAuth2PasswordBearer (提取 token)
↓
get_current_user (解碼 & 驗證 token)
↓
get_current_active_user (檢查帳號狀態)
↓
路由函式 (使用 user 物件) 
async def get_current_user(
    token: Annotated[str, Depends(oauth2_scheme)]
):
    user = fake_decode_token(token)
    if not user:
        raise HTTPException(
            status_code=401,
            detail="Invalid credentials",
            headers={"WWW-Authenticate": "Bearer"},
        )
    return user

async def get_current_active_user(
    user: Annotated[User, Depends(get_current_user)]
):
    if user.disabled:
        raise HTTPException(status_code=400, detail="Inactive user")
    return user

🪙 OAuth2 + JWT 完整實作

安裝：pip install pyjwt "pwdlib[argon2]"

import jwt
from datetime import datetime, timedelta, timezone
from pwdlib import PasswordHash

SECRET_KEY = "09d25e094faa..."  # openssl rand -hex 32
ALGORITHM = "HS256"
ACCESS_TOKEN_EXPIRE_MINUTES = 30

password_hash = PasswordHash.recommended()
DUMMY_HASH = password_hash.hash("dummy")  # 計時攻擊防護

def create_access_token(data: dict, expires_delta: timedelta | None = None):
    to_encode = data.copy()
    expire = datetime.now(timezone.utc) + (expires_delta or timedelta(minutes=15))
    to_encode.update({"exp": expire})
    return jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)

async def get_current_user(
    token: Annotated[str, Depends(oauth2_scheme)]
):
    credentials_exception = HTTPException(
        status_code=401, detail="Could not validate",
        headers={"WWW-Authenticate": "Bearer"},
    )
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
        username = payload.get("sub")
        if username is None: raise credentials_exception
    except jwt.InvalidTokenError:
        raise credentials_exception
    user = get_user(db, username)
    if user is None: raise credentials_exception
    return user

@app.post("/token")
async def login(
    form_data: Annotated[OAuth2PasswordRequestForm, Depends()]
):
    user = authenticate_user(db, form_data.username, form_data.password)
    if not user:
        raise HTTPException(status_code=401,
            headers={"WWW-Authenticate": "Bearer"})
    token = create_access_token(
        data={"sub": user.username},
        expires_delta=timedelta(minutes=30)
    )
    return {"access_token": token, "token_type": "bearer"}
⚠️
計時攻擊防護：當使用者不存在時，仍然要對 DUMMY_HASH 進行密碼驗證，確保回應時間一致。
永遠使用 secrets.compare_digest() 做字串比較，Python 的 == 會短路。

🎭 OAuth2 Scopes（進階）

使用 Security() 取代 Depends() 來宣告所需的 scopes。

from fastapi import Security
from fastapi.security import SecurityScopes

oauth2_scheme = OAuth2PasswordBearer(
    tokenUrl="token",
    scopes={
        "me": "讀取使用者資訊",
        "items": "讀取項目",
    }
)

async def get_current_user(
    security_scopes: SecurityScopes,
    token: Annotated[str, Depends(oauth2_scheme)]
):
    # security_scopes.scopes 收集依賴樹中所有要求的 scopes
    for scope in security_scopes.scopes:
        if scope not in token_data.scopes:
            raise HTTPException(status_code=401)
    return user

@app.get("/users/me/items/")
async def read_own_items(
    user: Annotated[User, Security(
        get_current_active_user,
        scopes=["items"]
    )]
):
    return [{"owner": user.username}]
ℹ️
SecurityScopes 自動收集整個依賴鏈中所有 Security(scopes=[...]) 宣告的 scopes。

🔒 HTTP Basic Auth

import secrets
from fastapi.security import HTTPBasic, HTTPBasicCredentials

security = HTTPBasic()

def get_current_username(
    credentials: Annotated[
        HTTPBasicCredentials, Depends(security)
    ]
):
    is_user = secrets.compare_digest(
        credentials.username.encode("utf8"),
        b"admin"
    )
    is_pass = secrets.compare_digest(
        credentials.password.encode("utf8"),
        b"secret"
    )
    if not (is_user and is_pass):
        raise HTTPException(
            status_code=401,
            headers={"WWW-Authenticate": "Basic"}
        )
    return credentials.username
⚠️
必須使用 secrets.compare_digest() — Python 的 == 運算子會短路（short-circuit），使攻擊者能透過回應時間差異推測正確的帳號密碼。
