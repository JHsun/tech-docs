🧠 模型更新

Opus 4.7
  • Opus 4.7 正式支援 — Claude Code 預設最強模型，需 v2.1.111+ (Agent SDK v0.2.111+)
  • Auto 模式預設啟用 — Max 訂閱者使用 Opus 4.7 不再需要 --enable-auto-mode
  • xhigh effort level — 新增最高 effort 級別，搭配互動式 /effort 滑桿即時切換
  • Adaptive Reasoning — Opus 4.7 & Sonnet 4.6 依 effort 動態分配 thinking tokens
Opus 4.6 增強
  • 1M context window — Max / Team / Enterprise 方案可用（預設仍為 200K）
  • 64K 預設 output tokens — 上限可達 128K（從 32K 倍增）
  • Prompt caching 12x 成本降低 — Agent SDK 修正快取斷點後大幅省錢
注意：Opus 4.7 需要 Agent SDK v0.2.111+。舊版 SDK 會降級至 Opus 4.6。

🖥 Computer Use 研究預覽

  • 原生 App 操控 — Claude 可開啟應用程式、點擊 UI 元素、輸入文字
  • 終端整合 — 從 CLI 測試 UI 變更，自動截圖驗證
  • 啟用方式：/mcp → 選擇 computer-use 切換
  • 適用場景 — UI 自動化測試、跨應用工作流、視覺驗證
研究預覽：功能仍在實驗階段，可能不穩定。僅建議用於開發/測試環境。

📅 Routines 雲端排程

  • Saved configs — 儲存 prompt + repos + connectors 組合
  • 雲端執行 — 在 Anthropic 基礎設施排程，無需本地機器
  • 管理介面：claude.ai/code/routines 或 CLI /schedule
  • Webhook 觸發 — 綁定 GitHub events（PR、issue、push）自動執行
  • Cron 排程 — 支援標準 cron 表達式定期執行

📱 Remote Control 遠端控制

  • 多端橋接 — 本地 session 同步至 claude.ai/code、iOS、Android
  • REST API + WebSocket — 即時雙向串流
  • 推播通知 — 手機接收任務完成/需確認通知
  • VS Code 整合：/remote-control 橋接 VS Code 至雲端
  • Extra Usage — 遠端控制客戶端現支援 /extra-usage

👥 Agent Teams 多代理協作

  • Team Lead 協調 — 多個 Claude Code instance 由主代理統籌分配任務
  • 獨立 context window — 每個子代理有自己的 200K context
  • Inter-agent 通訊 — 代理之間直接傳遞訊息，無需經由使用者
  • 啟動方式：--agents flag 或 /agents 管理
  • 背景代理：Ctrl+B 送至背景，Ctrl+F 終止全部

☁ Managed Agents 託管代理 Public Beta

核心特性（2026年4月8日 Public Beta）
  • 完全託管 — 在 Anthropic 基礎設施上執行，免維護
  • 沙盒執行 — 安全隔離環境，自動 checkpointing
  • Credentials 管理 — 安全注入 secrets、API keys
  • SSE Streaming — Server-Sent Events 即時串流進度
  • Scoped Permissions — 精細權限控制（allowedTools/deny 清單）
API 整合
  • E2E Tracing — 完整追蹤鏈，串接 TRACEPARENT / TRACESTATE
  • Resumable Sessions — 可斷線恢復、resume 先前 session
  • 定價：API token 費用 + $0.08/hour 計算時間
  • SDK 支援：TypeScript & Python claude-agent-sdk

  • OAuth RFC 9728 — 標準化 OAuth 授權流程
  • Computer Use MCP — 透過 /mcp 切換啟用

📦 Agent SDK 更新

  • 更名 — 正式更名為 claude-agent-sdk
  • Python SDK — 新增 Python 版本支援
  • 12x 成本降低 — 修正 prompt cache breakpoint 後大幅省錢
  • Opus 4.7 支援 — 需 v0.2.111+，舊版自動降級
  • 分散式追蹤 — 支援 TRACEPARENT / TRACESTATE 標頭
npm install claude-agent-sdk@latest # 或 pip install claude-agent-sdk

🛡 安全強化

  • macOS 危險路徑偵測 — 偵測操作 ~/Library、/System 等敏感路徑
  • Bash deny 包裝偵測 — 阻擋 env、sudo、watch 繞過 deny 規則
  • find 安全限制 — find -exec 和 -delete 不再自動核准
  • PID namespace 隔離 — Linux 上啟用程序命名空間隔離
  • 憑證清洗 — 子程序環境變數自動移除 credentials
企業注意：建議更新至 v2.1.100+ 以獲得完整安全修補。

🖥 全螢幕渲染 & Native Binary

TUI 全螢幕模式
  • /tui — 切換全螢幕渲染，告別終端閃爍
  • CLAUDE_CODE_NO_FLICKER — 環境變數永久啟用
  • /focus — 獨立 focus view，不再綁定 Ctrl+O
Native Binary 架構（v2.1.113）
  • 獨立執行檔 — 不依賴 Node.js runtime
  • 自動背景更新 — 無需手動 npm update
  • 安裝：curl -fsSL https://claude.ai/install.sh | bash
推薦：新安裝優先選擇 Native Binary，效能更佳且自動更新。
