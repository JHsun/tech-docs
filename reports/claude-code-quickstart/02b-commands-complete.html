⭐ 每日必用

/help列出所有可用命令
/init掃描專案產生 CLAUDE.md ★
/compact [指示]壓縮 context ★
/clear清除對話歷史 ★
/cost查看 token 使用統計 ★
/context可操作的 context 使用量建議 ★
/model [名稱]切換模型 ★
經驗法則：context 80%+ → /compact；切換任務 → /clear；auto-compact 在 95% 觸發但可能丟失細節。

📋 Session 管理

/resume [id]恢復先前 session
/rename [名稱]重新命名
/branch [名稱]分支當前 session（原 /fork，/fork 仍可用）
/rewind回退對話到某個點（含程式碼）
/recap回到 session 時提供上下文摘要
/export [檔名]匯出對話（或 --clipboard）
/exit退出 session
/branch vs /rewind：想試 → /branch；想撤銷 → /rewind。別名：/fork = /branch, /undo = /rewind
