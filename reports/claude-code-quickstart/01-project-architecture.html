🚀 快速開始 (2026)

1
安裝（推薦 native binary）
curl -fsSL https://claude.ai/install.sh | bash
或 npm：npm i -g @anthropic-ai/claude-code（需 Node 18+）
2
cd your-project && claude
3
執行 /init 自動產生 CLAUDE.md
4
在 .claude/commands/ 建立斜線命令
5
在 .mcp.json 設定 MCP 伺服器
6
依需求加入 skills、agents、hooks
注意：native binary 自動背景更新，不需 Node.js。npm 仍受支援但非首選。不要用 sudo npm install -g。

🔌 擴充類型 (Extension Types)

CLAUDE.md

永遠載入的專案記憶。每次對話自動讀取。

Skills

按需載入的 Markdown 知識 & 工作流。可透過斜線命令觸發或自動匹配。

MCP

外部工具連線協定。連接 GitHub、DB、Slack 等。支援 url & stdio。

Hooks

生命週期事件回呼。確定性執行，4 種 handler：command、http、prompt、agent。

Subagents

獨立隔離的子代理。自己的 context window，不繼承父對話。

Agent Teams

多代理協作（實驗性）。多個獨立 session 共享任務 & 訊息。

Plugins 是封裝層 — 將 skills + hooks + agents + MCP 打包成可安裝的單元。

⚙️ settings.json 結構

{ "permissions": { "allow": [...], "deny": [...] }, "hooks": { "PreToolUse": [{ "matcher": "Bash", "hooks": [{ "type": "command", "command": "./validate.sh" }] }], "PostToolUse": [{ "matcher": "Write|Edit|MultiEdit", "hooks": [{ "type": "command", "command": "npx prettier --write \"$CLAUDE_TOOL_INPUT_FILE_PATH\"" }] }] }, "env": { "MAX_THINKING_TOKENS": "10000" } }

🔗 .mcp.json 結構

{ "mcpServers": { "github": { "type": "stdio", "command": "npx", "args": ["-y", "@anthropic/mcp-github"], "env": { "GITHUB_TOKEN": "${GITHUB_TOKEN}" } }, "slack": { "type": "url", "url": "https://mcp.slack.com/sse" } } }
支援類型：stdio url

📝 CLAUDE.md 建議內容

# Project: My App ## Tech Stack - Next.js 14, TypeScript, Tailwind - Supabase for auth & database - tRPC API layer ## Conventions - Always write tests before code - Use conventional commits - Never commit directly to main ## Architecture - src/components — React components - src/services — Business logic - src/utils — Shared helpers ## Security - No secrets in code or logs - Validate all user inputs - Use parameterized queries only
無固定格式，保持簡潔。用 /init 自動產生。
