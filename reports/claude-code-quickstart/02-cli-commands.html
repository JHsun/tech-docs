⬇ 安裝方式

推薦：Native Binary（無需 Node.js）
# macOS / Linux curl -fsSL https://claude.ai/install.sh | bash # Windows (PowerShell) irm https://claude.ai/install.ps1 | iex # Homebrew brew install claude-code
替代：npm（需 Node.js 18+）
npm install -g @anthropic-ai/claude-code # ⚠ 絕對不要用 sudo npm install -g
自動更新：Native binary 自動背景更新。Homebrew/npm 需手動更新。
帳號需求：Pro ($20/月)、Max ($100-200/月)、Teams、Enterprise 或 Console。

🚩 CLI 啟動旗標 (claude [flags])

旗標說明範例
-p "prompt"Headless 模式（非互動），適合腳本/CIclaude -p "fix lint errors"
--model指定模型--model claude-sonnet-4-6
--permission-mode權限模式：default / plan / auto-accept--permission-mode plan
--allowedTools授權特定工具（headless 必用）--allowedTools "Read,Write,Bash(git *)"
--output-format輸出格式：text / json / stream-json--output-format stream-json
--append-system-prompt附加自訂系統提示（保留內建功能）--append-system-prompt "Always use TS"
--system-prompt完全替換系統提示（移除內建指令）
--resume / --continue恢復/繼續上次對話claude --resume
--verbose顯示詳細資訊（含思維過程）
--dangerously-skip-permissions跳過所有權限提示（僅限容器！）危險
--agents以 JSON 定義子代理
Headless 模式 (CI/CD)：claude -p "run tests" --allowedTools "Bash" --output-format stream-json

🧠 Extended Thinking & Plan Mode & Session 管理

Extended Thinking

預設開啟。Claude 在回覆前先內部推理。用 Ctrl+O 可看思維。

Adaptive Reasoning：Opus 4.6 & Sonnet 4.6 依 effort 動態分配思考 tokens。

適合：複雜架構、棘手 bug、多步驟規劃。

⚠ Thinking tokens 會計費。

Plan Mode

讓 Claude 分析 & 規劃但不執行修改。

# 啟動時 claude --permission-mode plan # Headless claude -p "重構 auth" --permission-mode plan # Session 內 /plan 或 Shift+Tab

Ctrl+G 在編輯器中修改 plan。

Session 管理
claude --resume恢復 session
claude --continue繼續最近 session
/clear清除歷史
/compact壓縮 context
/rename重新命名

• 每個 session 獨立
• 綁定當前目錄
• Git 分支切換後看新檔案但對話不變
• 修改前自動快照
• Auto-memory 跨 session 持久化

@ @ 引用 & 上下文注入

@filename引用檔案（fuzzy match）
@dir/引用整個目錄
@file#5-10引用特定行數
@server:resourceMCP 伺服器資源
