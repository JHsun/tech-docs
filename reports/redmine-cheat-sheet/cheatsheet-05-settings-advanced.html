版本庫設定（專案層級）
RedmineRepositories

Git

Redmine 需連結bare repository（非一般 working copy）

# 建立 bare repo
git clone --bare \
  /path/to/repo.git \
  /var/redmine/git/project.git

# 或建立 mirror
git clone --mirror \
  https://github.com/org/repo \
  /var/redmine/git/project.git

路徑填入：/var/redmine/git/project.git

Subversion

直接指定 SVN repository URL：

# 本地存取
file:///var/svn/project

# 遠端存取
https://svn.example.com/repos/project

可設帳號密碼進行認證。

Mercurial / CVS / Bazaar

均需指定本地 repository 路徑。

  • Mercurial：本地 hg repo 路徑
  • CVS：CVSROOT 路徑 + 模組名
  • Bazaar：本地 bzr repo 路徑
使用者對應：可在「版本庫」設定中將 commit 作者名對應到 Redmine 使用者，讓活動頁面和議題引用正確關聯。

接收郵件建立/更新議題
RedmineReceivingEmails

Redmine 可以接收郵件來自動建立議題或新增備註。支援三種接收方式：

方式 1：郵件轉發

郵件伺服器直接將郵件 pipe 給 Redmine 腳本：

| ruby /path/to/redmine/\
  extra/mail_handler/\
  rdm-mailhandler.rb \
  --url https://redmine.example.com \
  --key [API-KEY] \
  --project myproject

方式 2：IMAP 輪詢

rake redmine:email:receive_imap \
  RAILS_ENV=production \
  host=imap.example.com \
  port=993 ssl=1 \
  username=redmine \
  password=secret \
  project=myproject \
  tracker=bug \
  allow_override=project,tracker

方式 3：POP3 輪詢

rake redmine:email:receive_pop3 \
  RAILS_ENV=production \
  host=pop.example.com \
  username=redmine \
  password=secret \
  project=myproject

POP3 讀取後預設刪除郵件

郵件中設定議題屬性：可在郵件正文中使用特殊關鍵字（位於郵件開頭），例如：
Project: myproject　Tracker: Bug　Status: In Progress　Priority: High　Assigned to: john

Rake 維護任務
RedmineRake
常用批次任務
Rake 任務功能建議排程
redmine:send_reminders發送到期議題提醒郵件（可指定天數）每日
redmine:fetch_changesets從版本庫拉取新的 changesets每 10~30 分鐘
redmine:email:receive_imap從 IMAP 讀取郵件建立議題每 5~15 分鐘
redmine:email:receive_pop3從 POP3 讀取郵件建立議題每 5~15 分鐘
redmine:tokens:prune清除過期的認證令牌每日
redmine:watchers:prune清除指向已刪除使用者的關注記錄每週
redmine:attachments:prune清除未關聯的上傳附件每週
其他維護任務
Rake 任務功能
generate_secret_token生成新的 session 密鑰
db:encrypt / db:decrypt加密/解密 SCM 和 LDAP 密碼（資料庫層級）
redmine:attachments:move_to_subdirectories將附件從平面目錄移至子目錄結構（效能優化）
redmine:email:test[email]發送測試郵件以驗證 Email 設定
redmine:permissions列出所有可用權限及其歸屬模組
redmine:plugins列出已安裝的外掛
redmine:plugins:migrate執行外掛的資料庫遷移
redmine:plugins:assets複製外掛靜態資源到 public 目錄
所有 Rake 任務前綴：RAILS_ENV=production bundle exec rake [任務名]
