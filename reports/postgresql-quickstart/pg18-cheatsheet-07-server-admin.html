📦 安裝與初始化 (Ch16-17)

initdb 初始化資料目錄

# 建立新的資料庫叢集
initdb -D /var/lib/pgsql/18/data \
  --encoding=UTF8 \
  --locale=en_US.UTF-8 \
  --data-checksums \
  --wal-segsize=64      # WAL段大小(MB)
  --auth-local=peer \
  --auth-host=scram-sha-256
--data-checksums 強烈建議啟用，可偵測資料損壞

伺服器控制

# pg_ctl
pg_ctl start -D $PGDATA [-l logfile]
pg_ctl stop -D $PGDATA [-m smart|fast|immediate]
pg_ctl restart -D $PGDATA
pg_ctl reload -D $PGDATA   # 重新載入設定
pg_ctl status -D $PGDATA
pg_ctl promote -D $PGDATA  # 備援升為主庫

# systemd
systemctl start postgresql-18
systemctl stop postgresql-18
systemctl reload postgresql-18

停機模式

smart等所有客戶端斷線
fast中斷連線，正常關閉（預設）
immediate立即終止，需要恢復
⚙️ 設定檔案與參數管理 (Ch19)

主要設定檔

postgresql.conf主設定檔（所有 GUC 參數）
postgresql.auto.confALTER SYSTEM 寫入，優先於 postgresql.conf
pg_hba.conf客戶端認證設定
pg_ident.conf使用者名稱映射

設定參數方式

-- SQL 方式（寫入 postgresql.auto.conf）
ALTER SYSTEM SET shared_buffers = '2GB';
ALTER SYSTEM RESET shared_buffers;

-- 會話級設定
SET work_mem = '256MB';
SET LOCAL work_mem = '256MB';  -- 僅本交易
RESET work_mem;

-- 資料庫/角色級
ALTER DATABASE mydb SET work_mem = '128MB';
ALTER ROLE analyst SET work_mem = '512MB';

-- 查看設定
SHOW work_mem;
SHOW ALL;
SELECT * FROM pg_settings WHERE name = 'work_mem';

參數 context 類別

Context何時生效
internal編譯時決定，不可更改
postmaster需重啟伺服器
sighupreload 即生效
superuser-backend超級使用者新連線
backend新連線時
superuser超級使用者 SET
user任何使用者 SET

重新載入設定

pg_ctl reload -D $PGDATA
SELECT pg_reload_conf();
-- 或 kill -HUP postmaster_pid

-- 檢查 pending_restart
SELECT name, setting, pending_restart
FROM pg_settings
WHERE pending_restart;
🎛️ 關鍵 GUC 參數分類 (Ch19)

連線與認證

listen_addresses'localhost'監聽位址（'*' 全部）
port5432監聽埠
max_connections100最大連線數
superuser_reserved_connections3保留給超級使用者
password_encryptionscram-sha-256密碼加密方式
authentication_timeout1min認證超時

記憶體

shared_buffers128MB共享緩衝（建議 RAM 25%）
work_mem4MB排序/雜湊（每操作）
maintenance_work_mem64MBVACUUM/CREATE INDEX
effective_cache_size4GB預估 OS 快取（RAM 50-75%）
huge_pagestry使用大頁面
temp_buffers8MB每連線臨時表緩衝
hash_mem_multiplier2.0雜湊記憶體乘數

WAL

wal_levelreplicaWAL 等級（minimal/replica/logical）
max_wal_size1GB觸發 checkpoint 的 WAL 量
min_wal_size80MB保留 WAL 空間
wal_compressionoffWAL 壓縮（pglz/lz4/zstd）
wal_buffers-1WAL 緩衝（-1=自動）
synchronous_commiton同步提交（on/off/local/remote_*）

查詢規劃器

random_page_cost4.0隨機讀取成本（SSD→1.1）
seq_page_cost1.0循序讀取成本
effective_io_concurrency1I/O 並發（SSD→200）
default_statistics_target100統計取樣量
jitonJIT 編譯

平行處理

max_worker_processes8最大背景 worker
max_parallel_workers_per_gather2每次 Gather worker 數
max_parallel_workers8總平行 worker
max_parallel_maintenance_workers2維護操作平行度

日誌

logging_collectoroff啟用日誌收集器
log_directory'log'日誌目錄
log_min_duration_statement-1慢查詢閾值（ms，-1=關閉）
log_statement'none'記錄語句（none/ddl/mod/all）
log_connectionsoff記錄連線
log_disconnectionsoff記錄斷線
log_lock_waitsoff記錄鎖等待
log_temp_files-1記錄臨時檔案（KB）
log_checkpointson記錄 checkpoint
log_autovacuum_min_duration10min記錄慢 autovacuum

超時

statement_timeout0語句超時（0=關閉）
lock_timeout0鎖等待超時
idle_in_transaction_session_timeout0閒置交易超時
idle_session_timeout0閒置連線超時
tcp_keepalives_idle0TCP keepalive 間隔
🔐 客戶端認證 pg_hba.conf (Ch20)

格式

# TYPE  DATABASE  USER  ADDRESS  METHOD [OPTIONS]

# 本地 Unix socket
local   all       all                  peer
local   all       all                  scram-sha-256

# TCP/IP
host    all       all   127.0.0.1/32   scram-sha-256
host    all       all   ::1/128        scram-sha-256
host    all       all   0.0.0.0/0      scram-sha-256

# SSL 連線
hostssl mydb      myuser 10.0.0.0/8    scram-sha-256
hostnossl all     all   0.0.0.0/0      reject

# 複寫連線
host    replication replicator 10.0.0.0/8 scram-sha-256

認證方式

trust無條件允許（勿用於生產）
reject拒絕
scram-sha-256密碼（推薦）
md5MD5 密碼（舊式）
password明文密碼（勿用）
peerOS 使用者名稱匹配（本地）
identident 伺服器驗證
gssGSSAPI/Kerberos
sspiWindows SSPI
ldapLDAP 認證
certSSL 客戶端憑證
radiusRADIUS 認證
pamPAM 認證
規則從上到下匹配，第一個匹配的規則生效
👤 角色與權限管理 (Ch21)
-- 建立角色
CREATE ROLE name [ WITH
  LOGIN | NOLOGIN
  SUPERUSER | NOSUPERUSER
  CREATEDB | NOCREATEDB
  CREATEROLE | NOCREATEROLE
  INHERIT | NOINHERIT
  REPLICATION | NOREPLICATION
  BYPASSRLS | NOBYPASSRLS
  CONNECTION LIMIT connlimit
  PASSWORD 'xxx'
  VALID UNTIL '2025-12-31'
  IN ROLE role [, ...]
];

-- CREATE USER = CREATE ROLE LOGIN
CREATE USER app_user PASSWORD 'xxx';

-- 角色繼承
GRANT reader_group TO app_user;
REVOKE reader_group FROM app_user;

-- 修改角色
ALTER ROLE name SET ...;
DROP ROLE name;

預定義角色 (PG14+)

🗄️ 資料庫管理 (Ch22)
-- 建立資料庫
CREATE DATABASE mydb
  OWNER = myuser
  ENCODING = 'UTF8'
  LOCALE = 'en_US.UTF-8'
  TEMPLATE = template0
  CONNECTION LIMIT = 100
  TABLESPACE = my_ts;

-- 修改
ALTER DATABASE mydb OWNER TO newowner;
ALTER DATABASE mydb SET timezone = 'Asia/Taipei';
ALTER DATABASE mydb RENAME TO newname;

-- 刪除（需先斷開所有連線）
DROP DATABASE mydb [WITH (FORCE)];

模板資料庫

template1預設模板，可自訂
template0乾淨模板，用於不同 encoding/locale

🔒 SSL/TLS 設定

postgresql.conf

ssl = on
ssl_cert_file = 'server.crt'
ssl_key_file = 'server.key'
ssl_ca_file = 'root.crt'
ssl_min_protocol_version = 'TLSv1.3'
ssl_ciphers = 'HIGH:!aNULL'

pg_hba.conf 強制 SSL

hostssl all all 0.0.0.0/0 scram-sha-256
hostnossl all all 0.0.0.0/0 reject

客戶端連線

psql "host=db.example.com dbname=mydb \
  sslmode=verify-full \
  sslrootcert=/path/ca.crt"
sslmode說明
disable不用 SSL
allow先試非 SSL，失敗再用 SSL
prefer先試 SSL（預設）
require必須 SSL，不驗證憑證
verify-ca驗證 CA
verify-full驗證 CA + 主機名（推薦）
