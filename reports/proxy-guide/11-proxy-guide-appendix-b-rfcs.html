APPENDIX B · RFC 列表

Internet Standards Reference
本書引用的所有 RFC · 按主題分類 · ~70 個

本書直接或間接引用 70+ 個 RFC（Request for Comments，IETF 標準文件）。本附錄按 9 個主題分類，每個 RFC 含編號、標題、年份、狀態、本書章節參照。RFC 全文都在 rfc-editor.org 免費取得。

RFC 是 Internet 的真正規格 — 比任何技術書都權威，因為實作者照 RFC 寫。但 RFC 寫得枯燥難讀，初學者讀不下去。建議的讀法：先讀本書相關章節建立 mental model，再回頭讀 RFC 對應段落確認細節。RFC 是 reference，不是入門教材。下方 9 大分類涵蓋本書所有引用 RFC，每條附章節參照方便交叉閱讀。

如何讀 RFC

RFC 寫作風格非常正式（「The protocol MUST...」「SHALL NOT...」），對工程師來說不好讀。三個技巧：(1) 先看 Abstract + Introduction建立 mental model；(2) 看 examples 段落（通常有具體例子）；(3) 使用本書當 navigator — 本書某章卡關時去讀對應 RFC，反過來不要。RFC 是 reference，不是教材。

延伸資源：rfc-editor.org（官方 RFC repository）、datatracker.ietf.org（IETF working group + draft 追蹤）、httpwg.org（HTTP working group）、quicwg.org（QUIC working group）、tlswg.org（TLS working group）。
